콘텐츠 카테고리 제안

생활형 분쟁을 기본으로 두되, 후반부로 갈수록 도덕적 회색지대와 제3자 개입 비중을 늘린다.

부부 / 연인

외도 의심, 도박, 재산 은닉, 휴대폰 감시, 양가 가족 개입.

이웃

층간소음, 보복, 반려동물, 사생활 침해, 주차, 공동공간 점유.

직장

성과 가로채기, 책임 전가, 소문 유포, 부당한 야근 압박, 데이터 은폐.

가족 / 동업

간병비, 상속 예고, 공금 사용, 계약 위반, 신뢰 배신.

같은 관계에서 다른 사건이 나오는 예시

반복 플레이 설계를 한눈에 보여주는 샘플. 같은 캐릭터 성향이라도 사건 구조는 완전히 달라진다.

관계/캐릭터사건 A사건 B사건 C
부부 / A 회피형, B 직선형 공동 통장 돈이 도박으로 사라짐 외도 의심이 사실은 보험 상담 오해였음 A가 사이비 종교에 빠져 집안 재산을 이동
이웃 / A 예민형, B 뻔뻔형 층간소음과 보복 주차 반려견 사고 책임 공방 공용 복도 CCTV 사생활 침해 분쟁
상사-부하 / A 냉정형, B 피해자형 실적 가로채기 사내 비밀 문서 유출 의심 허위 보고와 책임 전가
💡
플레이어가 "이 캐릭터는 이런 성향이구나"를 기억할수록 몰입은 깊어지지만, 사건 자체는 계속 달라지므로 메타 학습만으로는 정답에 도달할 수 없다.

샘플 생성 사건

"부부" 관계에서 생성된 하나의 예시. 실제 게임에서는 같은 부부 템플릿에서도 전혀 다른 사건이 재생성된다.

Party A
민준 — 회피형, 체면 중시
  • 진실: 공동 통장에서 돈을 빼 도박에 사용
  • 오해 유도: 늦은 귀가의 이유를 외도처럼 보이게 숨김
  • 거짓말 전략: S0(전면 부정) → 계좌 내역 제시 시 S2(부분 인정) → S3(B가 먼저 의심했다)
  • 감정 곡선: 방어적 → 자신감 → 동요 → 체념
Party B
서연 — 직선형, 의심이 빠름
  • 진실: 휴대폰을 무단 열람했고 부모에게 과장 전달
  • 오해: 보험 상담 채팅을 외도 증거로 확신
  • 거짓말 전략: S0(외도라고 단정) → 채팅 전문 공개 시 S2(인정) → S4(감정 호소)
  • 감정 곡선: 분노 → 확신 → 혼란 → 슬픔
쟁점실제 진실지식 분면입증 수단책임 비율
공동 통장 무단 인출사실A만 앎E1 계좌 내역 (Hard)A 100%
도박 사용사실A만 앎E1 → E2 베팅 기록 (Hard)A 100%
외도거짓 (B의 오인)B만 앎(오해)E3+E4 조합 → Hard 격상A 20% / B 80%
휴대폰 무단 열람사실B만 앎E5 기기 접근 로그 (Hard)B 100%
보험 상담 (제3자 비밀)사실둘 다 모름E6 보험사 로그 (Hard) — E5 이후 잠금 해제반전 포인트

밸런스 원칙

언제나 해결 가능한 사건이어야 하지만, 항상 단순하지는 않아야 한다.

핵심 사실 1~2개는 확정 가능

플레이어가 아무리 못해도 사건의 축 하나는 파악할 수 있어야 좌절감이 줄어든다.

부가 쟁점은 회색지대 허용

모든 항목이 명쾌할 필요는 없다. 일부는 해결책의 지혜를 시험하는 용도로 남겨도 된다.

감정 과열은 정보 생산 장치

말싸움이 길어지는 것은 지루함이 아니라 실언과 모순 생산으로 이어져야 한다.
