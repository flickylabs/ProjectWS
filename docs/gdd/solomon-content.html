진행 구조

메인 모드
CaseMap — 7챕터 x 12스테이지

캔디크러시 스타일 스테이지 맵. 각 챕터 = 관계 유형 1개. SVG 베지어 곡선으로 노드 연결. 좌우 지그재그 배치.

stageMap 순서: spouse(1) → neighbor(2) → workplace(3) → partnership(4) → family(5) → tenant(6) → friend(7)

별 시스템: 1star(40+점) 2star(60+점) 3star(80+점)

해금: 챕터 순서대로 진행. 이전 챕터 일정 스테이지 클리어 시 다음 챕터 해금.

부가 시스템
리더보드 + 시즌 + 프로필

시즌 30일 단위 자동 계산. 로컬 리더보드(Phase 1). 스코어보드/명예의전당/통계 탭. 프로필 페이지에 명성 티어 표시.

점수 & 명성 티어

3축 평가

통찰 (insight)
사실 적중도

핵심 사실 정확도, 책임 비율 정확도. 모호 쟁점 보류는 패널티 없음.

권위 (authority)
절차 통제

법정 통제력 잔여량, 턴 효율성, 위법 증거 사용 여부, 비공개 약속 준수.

지혜 (wisdom)
판단 적절성

해결책 선택, 극단적 판결 회피, 모호한 쟁점의 적절한 보류.

총점 = insight + authority + wisdom + processBonus

Context Pack (ContextSeed 맥락 풀)

같은 인물쌍이 반복 등장할 때 사건 톤을 바꾸는 핵심 장치. 적은 비용으로 콘텐츠 다양성을 크게 늘린다.

명절 직전

가족 개입 증가, 체면 압박 극대화, 지출 스트레스. 부부/가족 관계에 최적.

인사평가 시즌

성과 압박, 책임 공방 격화, 문서 접근 민감. 직장 관계에 최적.

전세/월세 갱신

재정 불안 극대화, 계약 해석 충돌. 세입자/동업자 관계에 최적.

병원비 부담

재정 + 감정 이중 압박, 간병 분담 갈등. 가족/부부 관계에 최적.

시험/입학 직전

자녀 관련 긴장, 교육비 갈등, 부모 역할 충돌. 부부/가족에 최적.

장마철/재난

소음/누수/시설 갈등 격화, 공용 공간 마찰. 이웃 관계에 최적.

회사 감사 시즌

문서 추적 민감, 책임 소재 급부상, 내부 고발 위험. 직장에 최적.

이사 주간

공간/물건/보증금 갈등, 이별/시작의 감정. 이웃/세입자에 최적.

반려동물 치료

비용 분담, 돌봄 책임, 감정적 애착. 이웃/부부에 최적.

관계별 전용 미니 보드

"사건만 다르고 조사 방식은 늘 같다"는 느낌을 깨는 장치. 동일한 범용 컴포넌트에 관계별 기본 탭 구성만 달리 한다.

밸런스 원칙

앵커 진실은 반드시 확정 가능

플레이어가 아무리 못해도 사건의 축 하나는 파악할 수 있어야 좌절감이 줄어든다.

부가 쟁점은 회색지대 허용

후반 Stage에서는 부분 해결 사건이 등장한다. 판단 보류도 지혜로 인정한다.

감정 과열은 정보 생산 장치

말싸움이 길어지면 실언과 모순이 나와야 한다. 감정만 높고 정보는 없는 구간이 없도록.

압박과 신뢰의 밸런스

모든 사건에서 두 루트 모두 유효해야 한다. 압박 전용 사건이나 신뢰 전용 사건은 없다. 비율만 다를 뿐.

증거 정당성은 딜레마

위법 증거를 쓰면 사실을 밝힐 수 있지만 권위가 떨어진다. 항상 트레이드오프가 있어야 한다.

편향 디렉터는 1회 1요소

플레이어 습관 역이용은 한 판에 하나만. 많으면 부당함을 느낀다.

증인은 항상 편향

중립 증인도 무의식적 왜곡이 있을 수 있다. 증인 증언을 그대로 믿으면 점수가 떨어진다.

리소스는 전략적 선택

조사 토큰과 스킬 포인트가 제한적이므로 모든 증거를 조사하거나 모든 스킬을 사용할 수 없다. 선택과 집중이 필요.
