반복 등장하는 인물쌍(DuoSeed)의 생활 위에서 사건이 발생하고, 플레이어는 사실과 오해, 압박과 신뢰, 판결과 중재 사이를 가르며 '지혜'를 증명한다. 같은 인물, 매번 다른 사건으로 수백 가지 경우의 수를 만드는 Procedural Case Drama.
