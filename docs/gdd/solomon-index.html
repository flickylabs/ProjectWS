⚖️ 솔로몬 GDD v7.0

LLM 심문 추리 게임 · Procedural Case Drama — 반복 등장하는 인물쌍의 생활 위에서 사건이 발생하고, 플레이어는 사실과 오해, 압박과 신뢰, 판결과 중재 사이를 가르며 '지혜'를 증명하는 리플레이형 추리 게임. 이 GDD만으로 완전히 동일한 게임을 개발할 수 있도록 모든 기획, 타입 정의, 엔진 로직, 프롬프트 구조를 상세히 기술합니다.
