# Phase 1 대사 생성 — 세션별 추가 지시

gpt-phase1-reference.md와 함께 해당 세션의 추가 지시를 붙여서 GPT에 전달하세요.

---

## Session 1~4: spouse (부부)

```
## 이번 세션 추가 규칙

관계: 부부 (spouse)
- 호칭: 서로 "자기" (고정, 다른 호칭 금지)
- 상대에게: 반말 (~야, ~잖아, ~거야)
- 재판관에게 상대 언급: "제 남편이", "제 아내가"
- "~냐?" 금지 → "~야?"

예시:
- "자기, 그게 무슨 소리야. 나는 가족 때문에 그런 거야."
- "재판관님, 제 남편이 공동통장에서 몰래 빼갔습니다."
```

## Session 5~8: neighbor (이웃)

```
## 이번 세션 추가 규칙

관계: 이웃 (neighbor)
- 호칭: "~씨" 또는 이름+씨 (예: "김 씨", "박진수 씨")
  → 단, 캐릭터 대사에서는 이름 대신 "옆집 분", "윗층 분", "저 사람" 등 사용
- 상대에게: 존댓말 기본. 감정이 격해지면 반말 전환 가능
- 재판관에게 상대 언급: "옆집 분이", "저 사람이", "윗층 사람이"

예시:
- "재판관님, 옆집 분이 새벽마다 소음을 내서 도저히 잠을 잘 수가 없습니다."
- (격해진 후) "당신이 그러고도 할 말이 있어요?"
```

## Session 9~12: workplace (직장)

```
## 이번 세션 추가 규칙

관계: 직장 상하 (boss_employee)
- 호칭: 직급 호칭 (팀장님, 대리, 과장님 등) 또는 "~씨"
  → 캐릭터 대사에서는 "제 상사가", "제 부하직원이", "팀장이", "부장님이" 등 사용
- 상대에게: 존댓말 기본. 상사→부하는 반말 가능. 격해지면 반말 전환 가능
- 재판관에게 상대 언급: "제 팀장이", "제 부하직원이" 등 직급 기반

예시:
- "재판관님, 제 팀장이 업무 외 시간에 계속 연락을 합니다."
- (상사가 부하에게) "내가 언제 그런 식으로 지시했어?"
- (부하가 상사에게) "팀장님, 그건 사실과 다릅니다."
```

## Session 13~16: family (가족)

```
## 이번 세션 추가 규칙

관계: 가족 (family) — 형제/자매, 부모/자녀 등
- 호칭: 관계에 맞게 (형/누나/오빠/언니/동생 + 이름, 또는 엄마/아빠)
  → 사건 데이터의 캐릭터 나이와 성별로 관계를 파악하세요
- 상대에게: 반말 (가족 간)
- 재판관에게 상대 언급: "제 형이", "제 동생이", "제 누나가", "저희 어머니가" 등

예시:
- "재판관님, 제 누나가 부모님 통장에서 돈을 먼저 빼갔습니다."
- (누나→동생) "도현아, 네가 뭘 알아. 내가 얼마나 고생했는데."
- (동생→누나) "누나, 그건 네 맘대로 한 거잖아."

주의: A와 B의 나이/성별을 확인해서 형/누나/동생 등 관계를 정확히 반영하세요.
```

## Session 17~20: friend (친구)

```
## 이번 세션 추가 규칙

관계: 친구 (friend)
- 호칭: 이름 반말 (예: "수진아", "민호야")
  → 단, 캐릭터 대사에서 이름 대신 관계 호칭 사용: "걔가", "제 친구가"
- 상대에게: 반말 (친구 간)
- 재판관에게 상대 언급: "제 친구가", "걔가"

예시:
- "재판관님, 제 친구가 빌려간 돈을 안 갚고 있습니다."
- (친구끼리) "야, 그때 분명히 갚겠다고 했잖아."
- (친구끼리) "그건 네가 먼저 약속을 어긴 거야."
```

## Session 21~24: partnership (동업)

```
## 이번 세션 추가 규칙

관계: 동업자 (partnership)
- 호칭: "~씨" 또는 직함 (대표, 공동대표)
  → 캐릭터 대사에서: "제 동업자가", "공동대표가"
- 상대에게: 존댓말 기본. 격해지면 반말 가능
- 재판관에게 상대 언급: "제 동업자가", "공동대표가"

예시:
- "재판관님, 제 동업자가 공동 자금을 임의로 사용했습니다."
- (격해진 후) "당신이 그 돈 어디다 쓴 건지 지금 당장 말해요."
```

## Session 25~28: tenant (임대인/임차인)

```
## 이번 세션 추가 규칙

관계: 임대인/임차인 (tenant_landlord)
- 호칭: "집주인", "세입자", "임대인", "임차인" 또는 "~씨"
  → 캐릭터 대사에서: "집주인이", "세입자가" 등
- 상대에게: 존댓말 기본. 격해지면 반말 가능
- 재판관에게 상대 언급: "집주인이", "세입자가", "제 임대인이"

예시:
- "재판관님, 집주인이 보증금을 돌려주지 않고 있습니다."
- "재판관님, 세입자가 월세를 석 달째 안 내고 있습니다."
- (격해진 후) "당신이 그 집을 어떤 상태로 만들어놨는데요?"
```
