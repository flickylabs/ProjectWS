# 6. 채점 & 진행 시스템

## 3축 채점 시스템

판결 후 3가지 축으로 점수를 매기고, 평균이 최종 점수가 된다.

### 🔍 통찰 (Insight) — "진실을 얼마나 정확히 파악했는가"
- **기본 점수**: 각 쟁점의 사실 판단 정확도 × 쟁점 가중치
- **책임 비율 정확도**: 정답과의 차이에 따라 가감
- **모호한 쟁점 보류**: ambiguity=high인 쟁점을 보류하면 70% 인정 (단정지으면 0 or 100)
- **과정 보너스**:
  - 거짓말 붕괴 1회당 +3 (최대 +10)
  - 증거 발견 1회당 +3 (최대 +6)
  - 자유질문 적중 1회당 +2 (최대 +6)
  - 심층 truth 해금 1회당 +4 (최대 +8)
  - 상성 적중 1회당 +2 (최대 +6)
  - 질문 효율(전이율 30%+) +5

### ⚖️ 권위 (Authority) — "법정을 얼마나 잘 통제했는가"
- **기본 점수**: 70점 시작
- **법정 통제력**: 남은 통제력 × 5
- **위법 증거 사용**: -15 per evidence
- **비공개 약속 위반**: -20
- **턴 초과**: 20턴 이후부터 턴당 -2 (최대 -10)
- **과정 보너스**:
  - 효과적 증거 제시 +4 (최대 +8)
  - 필수 경로 충족 +3 (최대 +6)
  - 양측 모두 심문 +4
- **패널티**:
  - 반복 심문 -2 (최대 -6)
  - 근거 없는 거짓말 붕괴 -6 (최대 -6)
  - **끼어들기 허용 -3 (최대 -9)** ← 핵심 메커닉

### 💡 지혜 (Wisdom) — "판결이 얼마나 현명한가"
- **기본 점수**: 50점 시작
- **해결 방안 제시**: +20
- **극단적 판결**: 모든 책임을 한쪽에 몰면 -20
- **모호한 쟁점 적절 보류**: +10
- **과정 보너스**:
  - 비공개 보호 활용 +4 (최대 +8)
  - 토글 스킬 활용 +2 (최대 +6)
  - 자유질문 2회+ 적중 +3
  - 보너스 경로 충족 +2 (최대 +4)
- **패널티**:
  - 상성 미스 -1 (최대 -4)

### 보상 시스템
- **첫 사건 해결**: 스킬 토큰 ×3
- **일일 접속**: 조사 토큰 ×2
- **별 3개 달성**: 스킬 토큰 ×1

### 리소스
- **스킬 토큰 (⚡)**: 스킬 사용에 필요
- **조사 토큰 (🔍)**: 자유 질문, 증인 소환에 필요
- **법정 통제력**: 끼어들기 허용 시 감소, 0이 되면 강제 페이즈 전환

## V3 업데이트: Readiness 기반 판결 조건

턴 수 대신 **성과 기반**으로 판결 가능 여부를 결정합니다.

```
readinessScore = min(cracked, 2)×1 + min(resolved, 2)×2 + min(investigation, 2)×1
               + min(collapse, 2)×1 + min(hidden, 1)×1 + min(confession, 1)×2

일반 판결: 턴 8+ AND score 5+ AND 진행 쟁점 2+ AND 돌파 1+
조기 종료: 턴 6+ AND (고백 1+ OR resolved 2+ OR ...)
강제 판결: 턴 16 → forced_incomplete → "불충분 심리" 감점
```

강제 판결 시 ForcedVerdictBanner가 표시되며, 신뢰도 감점이 적용됩니다.
