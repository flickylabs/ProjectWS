# 3. 핵심 메커닉 상세

## Discovery 시스템 (v7.0 핵심 추가)

Phase 3~5에서 작동하는 4가지 능동 발견 메커닉. 플레이어가 단순히 "질문 → 답변"을 반복하는 것이 아니라, 진실을 **발견**하는 느낌을 주기 위한 시스템.

### 1. 진실 공방 (Truth Confrontation)
- **트리거**: A와 B가 같은 쟁점에서 서로 모순되는 주장을 할 때 자동 발생
- **예시**: A "280만원은 간병비" vs B "외도 자금이 분명해"
- **플레이어 선택**: believe_a / believe_b / both_partial / undetermined
- **효과**: 판단 결과가 이후 질문의 방향과 에이전트 반응에 영향

### 2. 증거 감별 (Evidence Appraisal)
- **트리거**: 같은 증거에 대해 2회 이상 조사 (원본 확인, 맥락 복원, 편집 탐지)
- **플레이어 판단**: trustworthy(신뢰) / partial(부분 신뢰) / suspicious(의심)
- **부분 신뢰 시**: 어떤 조사 결과를 신뢰할지 세부 선택 가능
- **함정 증거 보너스**: 위조 증거를 정확히 간파하면 추가 점수

### 3. 숨겨진 쟁점 발현 (Hidden Dispute Emergence)
처음에 보이지 않던 쟁점이 심문 과정에서 드러나는 시스템.
- **5가지 발현 경로**:
  1. 증거에서 → 새 증거 조사 시 연관 쟁점 발현
  2. 진실 공방에서 → 공방 판단 후 관련 숨겨진 쟁점 발현
  3. 증인에서 → 증인 증언이 새 쟁점을 언급
  4. 거짓말 붕괴에서 → 거짓말이 S5(침묵/재구성)까지 무너지면 관련 쟁점 발현
  5. 감정 폭발에서 → 격앙(explosive) 상태의 실수 자백으로 발현

### 4. 감정 전략 (Emotional Leverage)
- **4-Tier 감정 시스템**:
  ```
  침착 (0~30)  : 방어력 높음, 거짓말 전이 ×0.5, 실수 확률 0%
  동요 (30~60) : 균형 상태, 거짓말 전이 ×1.0, 실수 확률 0%
  격앙 (60~85) : 방어력 약함, 거짓말 전이 ×1.5, 실수 확률 30%
  셧다운 (85~100): 응답 거부, 2턴간 질문 불가, 공감으로 낮춰야 함
  ```
- **전략적 딜레마**: 밀수록 전이가 빨라지지만, 너무 밀면 셧다운되어 아무 것도 못 함

## 증거 시스템

### 증거 유형
- **Hard Evidence** (기관 발행): 은행 거래내역, 관공서 문서, CCTV — 1개만으로 쟁점 붕괴 가능
- **Soft Evidence** (개인 자료): 카톡 캡처, 녹음, 사진 — 2개 이상 모이면 Hard로 업그레이드

### 4축 감별
증거 하나에 대해 4가지 축으로 평가:
1. **신뢰성** (trustworthiness) — 내용이 사실과 부합하는가
2. **완전성** (completeness) — 빠진 부분이 없는가
3. **출처 신빙성** (source credibility) — 제공자가 믿을 만한가
4. **적법성** (legitimacy) — 합법적으로 수집된 증거인가

### 3단계 조사
1. **원본 확인** (verify_original) — 증거 자체를 확인
2. **맥락 복원** (restore_context) — 증거가 나온 상황을 파악
3. **편집 탐지** (detect_editing) — 변조 여부 확인

### 적법성 판단
- `lawful` → 자유롭게 사용 가능
- `gray` → 사용은 가능하나 권위 감점 위험
- `unlawful` → 사용 시 -15 권위 패널티
- **비공개 보호**: 상대가 "비공개로 해주세요"라고 요청한 증거를 판결에 쓰면 -20 패널티

## 질문 시스템

### 3가지 질문 유형과 효과
| 유형 | 아이콘 | 주요 효과 | 감정 변화 |
|------|--------|-----------|-----------|
| 사실 추궁 | 🔵 | 쟁점 직접 공격, 거짓말 전이 유도 | 감정 상승 |
| 동기 탐색 | 🟣 | 숨겨진 동기 탐색, 맥락 확보 | 약간 상승 |
| 공감 접근 | 🩷 | 신뢰 +12, 감정 하강 | 감정 하강 |

### 상성 (Affinity) 시스템
각 쟁점의 거짓말 유형에 따라 효과적인 질문이 다르다:
- 완전 부정(LT-1) → 사실 추궁이 효과적
- 감정 은폐(LT-5) → 공감 접근이 효과적
- 책임 전가(LT-4) → 동기 탐색이 효과적
- 상성이 맞으면 전이 확률 보너스, 안 맞으면 패널티
