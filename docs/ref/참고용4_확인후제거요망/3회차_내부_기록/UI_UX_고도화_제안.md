# UI/UX 고도화 제안

> 현재 구현 상태 기반으로, 게임 경험을 크게 개선할 수 있는 방향을 제안합니다.

---

## A. 홈화면

### A-1. 일일 보상 팝업 (로그인 보상)
- 접속 시 일일 보상 모달 표시 (7일 연속 패턴)
- Day 1: 🔍×1, Day 2: ⚡×1, Day 3: 🔍×2, ... Day 7: 특별 칭호
- 현재 rewardEngine에 daily_login만 있음 → 스트릭 시각화

### A-2. 배너 슬라이드
- 홈화면 상단에 이벤트/업데이트 배너 자동 슬라이드
- WebAdmin에서 배너 이미지+링크 관리
- 현재 공지와 별개의 비주얼 요소

### A-3. 빠른 시작 버튼
- "이어하기" (진행 중 사건), "추천 사건" (플레이 안 한 것 중 난이도 맞는 것)
- 현재는 항상 CaseMap으로만 진입

---

## B. 게임 플레이

### B-1. NPC 감정 시각화 강화
- 현재: behaviorHint 텍스트만 표시
- 개선: NPC 아바타 옆에 감정 아이콘 (😤😰😢😠😌) 실시간 변화
- 감정 변화 시 아이콘에 짧은 애니메이션 (흔들림, 커짐)

### B-2. 쟁점 상태 진행 바
- 현재: 쟁점별 lie state를 직접 보여주지 않음
- 개선: 각 쟁점 옆에 S0~S5 5단계 진행 바 (회색→노란→빨강→초록)
- 플레이어가 "얼마나 가까워졌는지" 체감 가능

### B-3. 증거 연결 지도
- 현재: 증거 목록만 나열
- 개선: 쟁점 ↔ 증거 연결을 시각적 맵으로 표시
- 어떤 증거가 어떤 쟁점을 증명하는지 한눈에 파악

### B-4. 타임라인 뷰
- 심문 히스토리를 시간순으로 시각화
- "이 쟁점에서 3번 질문했고, S1에서 S3으로 전이됨" 같은 흐름
- 판결 전 복습용

### B-5. 스킬 쿨다운 시각화
- 현재: 스킬 포인트/법정 통제력 숫자만 표시
- 개선: 스킬 버튼에 남은 횟수/쿨다운을 직관적으로 표시

---

## C. 결과/판결

### C-1. 판결 리플레이
- 판결 후 "핵심 장면 다시 보기" — 중요 전이 순간을 하이라이트
- "이 질문이 결정적이었습니다" 표시

### C-2. 점수 분해 시각화
- 현재: 3축 점수만 표시
- 개선: "무엇이 점수를 올렸고, 무엇이 깎았는지" 상세 분해
- 상성 보너스/패널티, 경로 커버리지, 심층 해금 등

### C-3. 진실 공개 연출
- 판결 후 anchorTruth 공개 시 드라마틱한 연출
- "사실 280만원은..." → 텍스트 타이핑 + 배경 전환

### C-4. SNS 공유
- 판결 결과를 이미지로 생성하여 공유
- "솔로몬 게임에서 통찰 95점을 받았습니다!"

---

## D. 프로필/소셜

### D-1. 프로필 카드
- 이모지 아바타 + 닉네임 + 티어 + 대표 칭호를 카드 형태로
- 리더보드에서 다른 플레이어 카드도 볼 수 있게

### D-2. 아바타 커스터마이징
- 이모지 선택 외에 프레임/배경색 선택
- 티어별 프레임 자동 적용 (수습=회색, 솔로몬=금색)

### D-3. 친구 시스템 (향후)
- 친구 추가 → 친구의 플레이 기록 비교
- "이 사건에서 친구는 어떤 판결을?" 호기심 유발

---

## E. 설정 확장

### E-1. 언어 설정
- 현재 한국어 only → 영어/일본어 준비

### E-2. 알림 설정
- 자동 충전 완료 알림, 시즌 종료 알림, 새 공지 알림
- Push 알림 (PWA/앱 전환 시)

### E-3. 데이터 관리
- 게임 데이터 백업/복원 (JSON 내보내기/가져오기)
- 계정 연동 (Supabase 전환 시)

### E-4. 접근성
- 글꼴 크기 조정
- 고대비 모드
- 색약 모드 (빨강/초록 대신 모양으로 구분)

## 구현 우선순위

- C-2 점수 분해 시각화 (v2 스코어링과 연계)

- A-1 일일 보상 스트릭
- B-3 증거 연결 지도
- C-3 진실 공개 연출
- D-1 프로필 카드
- D-2 아바타 커스터마이징

### 중기 (1~2개월)
- A-2 배너 슬라이드
- B-4 타임라인 뷰
- C-1 판결 리플레이
- C-4 SNS 공유
- E-2 알림 설정
- E-3 데이터 관리
