# 호칭(callTerms) 검토 이슈 목록 — 사건별 상세

> Phase 1 스크립트 + AI 생성용 프롬프트 동시 영향. 수정 전 방향 합의 필요.
>
> **AI 프롬프트 영향 원리**: `llmSpeechGuide.ts`의 `getMyCall()`, `getJudgeReference()`, `getAngryCall()`이
> 사건 JSON의 `partyA.callTerms` / `partyB.callTerms`를 그대로 읽음.
> → **callTerms가 잘못되면 Phase 1 스크립트와 AI 프롬프트가 동시에 같은 문제를 가짐.**
>
> 추가로, `llmSpeechGuide.ts`의 `isInformalRelation()`이 `family`를 일괄 반말 관계로 처리.
> → 부모-자녀 관계에서 자녀가 부모에게 반말하는 문제도 있음.
