## 각방 사유 — 시댁 갈등의 핵심

시부모가 돌아가신 후, 재산(집+토지)을 **형(이성호)이 혼자 상속받아 전부 탕진**한 사실이 드러남.
- B(이준호)는 이미 알고 있었지만 A에게 말하지 않음
- A가 뒤늦게 알게 됨 → "왜 나한테 말 안 했어?" → 대폭발 → 각방
- 각방 = 시댁 갈등의 결과이자, B가 시댁 관련 얘기를 못 꺼내는 구조적 이유

**게임 내 노출 경로:**
- Phase1 또는 심문 초반에 A가 "각방 쓴 지 8개월째"를 자연스럽게 언급
- "왜 각방이죠?" → A: "시댁 때문에요" → 시댁 갈등 자연 노출
- B의 침묵 동기: "시댁 얘기 꺼내면 각방도 모자라 이혼이니까"

## 숨긴 행동 정리

### A(박지연)가 숨기는 것
1. 신혼 초부터 B 계좌를 몰래 확인해옴 (사생활 침해)
2. 위임장을 조작해서 공동 적금 2,000만원 해지 (**범죄**)
3. 해지한 돈을 투자방에 넣었다가 사기 당함 (전액 손실)

### B(이준호)가 숨기는 것
1. 개인 비자금 운용 (아내 몰래)
2. 비자금에서 3,000만원을 형에게 현금 전달 (아내 동의 없이)
3. 오피스텔 방문 = 조카 돌봄 (아내에게 설명 안 함)
4. 시부모 재산 탕진 사실을 이미 알고 있었음 (아내에게 숨김)

## 핵심 드라마 포인트

1. **A가 계좌를 훔쳐봤다는 사실이 터지는 순간** — B: "그걸 어떻게 알아?" → 양쪽 다 침묵 → 재판관이 추궁
2. **위임장 조작이 밝혀지는 순간** — B도 충격, A는 벌받을 각오
3. **경비 증언으로 "여자 혼자 사는 집이 없다"** — 외도 서사에 첫 균열
4. **참고서 영수증이 부각되는 순간** — "중학생이 왜?" → 조카 연결
5. **형의 개인회생 사실** — B가 현금으로 전달할 수밖에 없었던 이유
