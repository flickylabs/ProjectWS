# CT → 스레드 B: Home UX 재설계

> 발신: CT
> 수신: 스레드 B (PC UI)
> 일시: 2026-04-09
> 중요도: 최우선 — Home 구조 변경

---

## 문제

현재 Home에 너무 많은 것이 한 화면에 있습니다. 사건 카테고리, 프리뷰 카드, 유저 정보, 리더보드, 재판관, 캠페인, 리소스가 전부 한 페이지에 펼쳐져 있어서 복잡하고 세련되지 않습니다.

## 원칙

**Home은 심플하고 세련되게. Depth를 두어 단계적으로 들어가는 구조.**

---

## Home 화면 구조

Home에는 **핵심 진입점만** 표시합니다:

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│                      ⚖️                               │
│                    솔로몬                              │
│          현대판 솔로몬이 되어 판결을 내려보세요!         │
│                                                      │
│                                                      │
│        ┌──────────────┐  ┌──────────────┐            │
│        │              │  │              │            │
│        │   일반 모드   │  │   시즌 모드   │            │
│        │              │  │              │            │
│        └──────────────┘  └──────────────┘            │
│                                                      │
│                                                      │
│     [내 정보]  [리더보드]  [판결 기록]  [설정]          │
│                                                      │
│                                                      │
│                              ● AI 연결됨              │
└──────────────────────────────────────────────────────┘
```

**Home에 없어야 할 것:**
- 사건 카테고리 버튼 나열
- 사건 프리뷰 카드
- 유저 상세 정보 (명성, 리소스 수치 등)
- 재판관 성향 상세
- 캠페인 진행 상태

이것들은 전부 **다음 Depth에서** 보여줍니다.

## 모바일 참고 — Depth 구조만

모바일(`src/app/App.tsx`)의 Depth:
```
Home
├── 사건 시작 → SessionSelect → CaseMap → 게임
├── 리더보드 → LeaderboardScreen
├── 내 기록 → HistoryPanel
├── 프로필 → ProfilePage
└── 설정 → SettingsPanel
```

PC에서의 Depth:
```
Home
├── 일반 모드 → 10개 세션 → 사건 리스트 → 게임
├── 시즌 모드 → 시즌 사건 리스트 → 게임
├── 내 정보 → 프로필 + 재판관 성향 + 리소스
├── 리더보드 → 리더보드 화면
├── 판결 기록 → 과거 판결 목록
└── 설정 → BGM, 공지, 우편 등
```

**항목은 모바일과 동일하거나 더 정리됨. UI는 PC에 특화.**

## DEV 필터 위치

기존 CaseBrowser 내 DEV 필터는 **일반 모드의 사건 리스트 화면**에 유지합니다. 세션을 선택한 후 사건 리스트에서 `[DEV: 완성 케이스만]` 토글.

---

## 현재 서버 이슈

localhost가 빙글빙글 돌며 화면이 안 나오는 문제가 있습니다. Home UX 재설계 전에 이 문제를 먼저 확인해 주세요:
- `npm run dev:pc` 실행 후 콘솔에 에러가 있는지
- 브라우저 콘솔에 무한 루프나 렌더링 에러가 있는지
- 최근 수정에서 무한 re-render를 유발하는 코드가 있는지

---

## 작업 순서

1. **localhost 로딩 문제 해결** — 화면이 안 나오면 아무것도 확인 불가
2. **Home 심플화** — 현재 컨텐츠 대부분 제거, 위 구조대로 재배치
3. **일반 모드 Depth** — 10세션 → 사건 리스트 → 게임 시작
4. **시즌 모드 Depth** — 시즌 사건 리스트 → 게임 시작
5. **내 정보 / 리더보드 / 판결 기록 / 설정** — PC 전용 UI로 각 화면 구성
