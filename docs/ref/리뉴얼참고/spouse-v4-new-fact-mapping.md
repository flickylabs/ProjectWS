---

## 읽는 법

- **NEW FACT** = 이전 State에서는 절대 나오지 않던 구체적 사실
- **질문 유형별**: 같은 State에서도 사실추궁/동기탐색/공감접근이 다른 조각을 줌
- **Truth Throttle**: S0~S1은 구체 명사 금지, S2는 범위 수준, S3+ 구체 허용
- **캐릭터 심리**: A는 남편의 "숨기는 버릇"에 실망/분노. B는 아내가 "돈 앞에서 가족을 자르는 것"에 실망.

## 증거 제시 응답 — 핵심 원칙

증거는 심문으로 얻을 수 없는 **확정 사실**을 제공

| 증거 | 적절한 대상 | 부적절한 대상 반응 |
|------|-----------|-----------------|
| e-1 영수증 묶음 | B → "그건... 제가 산 겁니다" | A → "제가 남편 차에서 찾은 겁니다. 남편한테 물어보세요" |
| e-2 블랙박스 GPS | B → "그 주소에 간 건 맞습니다" | A → "이걸 제가 확인한 겁니다" |
| e-3 통화기록 | B → "그 번호는... 설명드리겠습니다" | A → "매일 새벽에 이 번호로 전화합니다. 누군지 물어보세요" |
| e-4 형 문자 | B → "형한테 온 문자 맞습니다" | A → "이건 처음 봅니다. 이게 뭔 내용입니까?" |
| e-5 B 계좌 출금 | B → "비자금에서 뺀 건 맞습니다" | A → "이건 제가 이미 알고 있었습니다" (계좌 감시 노출 위험) |
| e-6 투자방 카톡 | A → "이건... 제가 보낸 겁니다" | B → "이건 처음 봅니다. 아내가 이런 걸 했습니까?" |
| e-7 적금 해지 서류 | A → "제가 해지했습니다" | B → "제 동의 없이 해지됐습니다. 위임장은 제가 쓴 게 아닙니다" |
