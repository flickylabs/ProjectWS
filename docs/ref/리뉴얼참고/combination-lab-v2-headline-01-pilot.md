## 1. 파일럿 목표

`headline-01`은 아래 4축이 명확해 조합 실험실 파일럿으로 적합하다.

- 폭로 영상의 공익성 vs 편집성
- 실제 위생 위반의 정도
- 무상 협조 요구와 사후 회복 패키지
- 점주 측의 보복성 신상 유포

기존 증거:

- `e-1` 원본 촬영본과 최종 업로드본 비교
- `e-2` 주방 CCTV와 위생 체크 사진
- `e-3` 구청 위생 시정 메일과 재점검 기록
- `e-4` 사전 협조 DM과 사후 회복 패키지 견적
- `e-5` 점주 단체방 공유 캡처와 삭제 복구본
- `e-6` 평점 하락 타임라인과 플랫폼 확산 로그

### Step 2

### Step 3

- 결과 로그/히스토리 카드 추가

## 6. 기대 플레이 감각

이 파일럿이 성공하면 플레이어는 다음 경험을 하게 된다.

- 증거를 단순히 순서대로 내는 게 아니라, 서로 맞물려 해석한다
- 대사에서 뽑은 노트가 실제 추리 자산이 된다
- 모든 조합이 정답은 아니어서 실험의 긴장이 생긴다
- 의외의 조합으로 새로운 질문과 해석이 열린다
