# 세션 1: spouse-01 보충 + family-01 Tell/Beat

## 프로젝트 배경

**Solomon**은 LLM 기반 법정 심문 게임입니다. 플레이어는 재판관으로서 양측 당사자(AI NPC)를 심문하여 진실을 밝힙니다.

### 핵심 아키텍처
- **엔진이 의미를 결정하고, LLM은 연기만 합니다.**
- 매 턴: `ResponseBlueprint` 생성 → `stance/defenseMode/allowedAtoms` 결정 → LLM이 1~3문장 응답
- NPC는 `lieState` 상태 머신(S0 완전부정 → S5 시인)을 따릅니다
- 질문 유형: `fact_pursuit`(사실추궁), `motive_search`(동기탐색), `empathy_approach`(공감접근), `evidence_present`(증거제시)

### V3 게임 루프 (이번 작업의 맥락)
1차 요청에서 만든 V3 시스템의 핵심:
- **DossierCard**: 증거 2개를 묶은 플레이어블 카드 → challenge 질문으로 공격
- **StateUnlockAtom**: state 진입 시 해금되는 새 사실 (S2~S5)
- **이벤트**: 모순 감지/끼어들기/감정 폭발 → 플레이어 선택지 2개
- **TransitionBeat**: lieState 전이 순간 강제 삽입하는 사전 작성 대사 (LLM 불사용)
- **ExecutableVerbalTell**: NPC의 말버릇 규칙 (어휘 훅 + 문장 구조 + 빈도)
- **BeatScript**: stance별 사전 작성 핵심 대사 (fallback + 연출용)

- **윤서아(a)**: 42세 프리랜서 보험설계사, archetype: victim_cosplay. 부모 예금 1,800만원 선이체
- **윤도현(b)**: 38세 냉동설비 정비사, archetype: cold_logic. 간병비 분담 3개월 지연

**쟁점 5개**:
- d-1: 서아의 부모 예금 선이체
- d-2: 도현의 간병비 분담 지연
- d-3: 아버지 수첩 메모는 상속 예고인가
- d-4: 첫 달 간병비는 전액 형제 사비로만 감당해야 했는가
- d-5: 공동 기록 원칙의 쌍방 위반

**증거 6개**:
- e-1: 부모 관리계좌 거래내역 + 서아 개인계좌 입금확인서 (hard)
- e-2: 서아 휴대폰 메모 복원본 + 자동이체 실패 알림 (soft)
- e-3: 가족 단톡 + 간병 스케줄표 원본 (soft)
- e-4: 2021년 아버지 수첩 메모 사진 (soft, **함정 증거** — 잘린/재촬영)
- e-5: 상속 상담 변호사 확인서 + 수첩 전체 스캔본 (hard)
- e-6: 병원 사회복지사 장기요양 경감 신청 기록 (hard)

### Tell 6개

**서아(a)** — victim_cosplay:
- `sacrifice_rollcall`: 궁지에 몰리면 자기가 한 간병 일정을 나열. appliesWhen: cornered, shame. shape: enumeration. cadence: every_turn.
- `tear_brake`: 목이 잠기며 "저 아니었으면 무너졌을 거예요" 반복. appliesWhen: emotional, hurt. shape: self_reference. cadence: once_every_2_turns.
- `echo_blame`: "그러면 당신은 어떻게 했겠어요?" 되물음. appliesWhen: cornered, defensive. shape: question_end. cadence: max_once_per_turn. requiresSecondSentence: true.

**도현(b)** — cold_logic:
- `receipt_stack`: 날짜/금액을 빠르게 나열하며 방어. appliesWhen: lying, avoiding. shape: number_first. cadence: once_every_2_turns. numericGranularity: exact_amount_or_minute.
- `clipped_denial`: "그건 아닌데요", "엄밀히 말하면 늦은 거지" 짧은 부정. appliesWhen: lying, defensive. cadence: every_turn.
- `dry_sarcasm`: "저는 돈이 없고 몸뚱이만 있으니까요" 건조한 비아냥. appliesWhen: shame, hurt. shape: self_reference. cadence: on_trigger_only.
