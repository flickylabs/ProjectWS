## 체크 A: S2 대사 다양성 (결정2 관련)

**질문**: S2에서 hedge beat만으로 충분한 다양성이 제공되는가?

**결과**:
- spouse-01 기준 S2 사용 가능 hedge beat: 쟁점별 2개씩 × 5쟁점 = 10개

- 20턴 중 S2 체류 3-4턴 → 같은 대사 반복 없음
- LLM이 alternatives와 slot material을 조합하여 자연스러운 변주 생성

**판정: PASS — 충분한 다양성 확인**

**83건 확장 시 권장**: 쟁점당 최소 2개 hedge beat + alternatives 2개 유지
