### 최우선 검증 항목 (이번 핵심)

**A. 화자 시점/호칭 — 이번에 가장 중요한 항목이야:**
- 재판관이 당사자를 언급할 때 → "한지석 씨", "오세린 씨" (이름+씨)
- 재판관이 절대 "제 아내", "제 남편" 사용 금지
- 당사자A → 재판관에게 B 언급: toJudge 값 ("제 아내가~")
- 당사자A → B에게 직접: toPartner 값 ("자기야~")
- 조사 정합성: "제 아내**가**"(✅), "제 아내**이**"(❌)
- 증인: addressA/addressB 값 사용

**B. 한국어 자연스러움:**
- 번역체 9패턴 0건
- "사전 상의/협의" 0건 (S0-S2 범위)
- 같은 표현 2턴 연속 0건
- 조사 오류 0건

**C. 몰입감/개연성:**
- archetype에 맞는 말투 차별화
- tell이 자연스럽게 녹아 있는지 (기계적 삽입 아닌지)
- S0→S5 감정 변화가 자연스러운 곡선
- 이전 발언과 모순 없음

### PASS 기준
- **CRITICAL 0건, FAIL 0건, WARN 5건 이하** = PASS
- 하나라도 CRITICAL이 나오면 즉시 보고 후 수정 → 재테스트
