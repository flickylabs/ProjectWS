## 1. 프로젝트 배경

### 솔로몬 게임이란
한국 민사 분쟁 재판 시뮬레이션 게임. 플레이어가 **재판관**이 되어 양측 NPC(당사자)를 심문한다.
- 한국어 전용 게임
- NPC 대사는 LLM이 실시간 생성
- **재판관 질문**은 미리 작성된 템플릿에서 선택

### DossierCard(증거 카드)란
플레이어가 수집한 증거를 묶어 NPC에게 **결정적 질문**을 던지는 시스템.
- 각 카드에는 **3개의 질문**이 있음
- NPC의 lieState(거짓말 상태)가 올라갈수록 더 날카로운 질문이 해금됨
  - q1: 처음부터 해금 (탐색적)
  - q2: S2에서 해금 (구체적)
  - q3: S3에서 해금 (압박적)

### 질문의 화자
**재판관**이 NPC에게 던지는 질문이다.

### friend-01 사건 개요
친구 여행 정산 분쟁.
- **서도윤** (partyA): 여행 정산 주도자. 보증금 환급을 늦게 알리고 택시비를 일방 상계한 혐의.
- **정하린** (partyB): 정산 이의 제기자. 중간 캡처를 근거로 SNS 공개 비난을 한 혐의.
- **김수아**: 공통 친구(제3자). 양쪽에서 먼저 이야기를 들음.

## 2. 문제점

**전 18건이 반말/당신체**로 작성되어 있음. 한국 법정에서 재판관은 절대 "당신"을 쓰지 않는다.

현재:
```
"서도윤, 당신은 18만4천원이 최종액이 아니라고 말합니다. 그렇다면..."
"정하린, 당신이 본 18만4천원 캡처는 자동분배 완료 전 화면입니다."
```

이래야 함:
```
"서도윤 씨, 이 18만4천원이 최종 정산액이 아니라는 입장이십니까? ..."
"정하린 씨, 이 캡처가 자동분배 완료 전 화면이라는 점을 알고 계셨습니까?"
```

## 3. 교정 규칙

### 호칭 규칙 (절대 준수)
| 현재 (잘못) | 교정 후 |
|------------|---------|
| "서도윤" (이름만) | "서도윤 씨" |
| "정하린" (이름만) | "정하린 씨" |
| "도윤" (줄임말) | "서도윤 씨" |
| "하린" (줄임말) | "정하린 씨" |
| "김수아" | "김수아 씨" (제3자도 씨 사용) |
| "당신", "당신이", "당신은", "당신도" | 삭제하고 문장 재구성 |

### 종결어미 (재판관 격식체)
| 현재 (잘못) | 교정 후 |
|------------|---------|
| "~지요?" | "~입니까?" 또는 "~하십시오" |
| "~겁니까?" | "~입니까?" |
| "~잖습니까?" | "~지 않습니까?" 또는 "~하십시오" |
| "~아닙니까?" | 허용 (격식체 반문) |
| "~아닌가요?" | "~아닙니까?" (해요체→격식체) |

### 유죄 전제 금지
```
나쁜 예: "왜 규칙을 어겼습니까?" (위반 전제)
좋은 예: "이 규칙이 지켜지지 않은 이유를 설명하십시오."

나쁜 예: "체면 아닙니까?" (동기 단정)
좋은 예: "이 분쟁을 유지하는 이유가 무엇입니까?"
```

### 보존해야 할 것
- 각 질문의 **attackVector** (authenticity/context/legality/identity)
- 각 질문의 **requiredLieState** (없음/S2/S3)
- 각 질문의 **핵심 논점** (무엇을 추궁하는지)
- **q1→q2→q3 압박 상승 구조**

## 4. 품질 기준

### 좋은 예시 (같은 게임의 neighbor-01 사건)
```
q1 (탐색): "정한결 씨, 관리사무소를 거치지 않고 개인 휴대폰을 세운 이유가 무엇입니까?"
q2 (구체): "점검 보고서가 세대 직접 원인을 적지 않는데도 왜 주연 세대라고 단정했습니까?"
q3 (압박): "복도 장면과 천장 누수를 한 원인으로 묶은 근거를, 추정이 아닌 사실만으로 설명해 보십시오."
```

특징:
- "씨" 호칭 일관
- "당신" 미사용
- q1→q2→q3 압박 상승
- 유죄 전제 없이 논리적 질문
- 자연스러운 한국어

### 금지 표현
| 금지 | 이유 |
|------|------|
| "당신" / "당신이" / "당신은" / "당신도" | 재판관 어법 위반 |
| "~지요?" | 해요체 혼용 |
| "~아닌가요" | 해요체 |
| "체면 아닙니까" | 동기 단정 |
| "해당 건에 대해서는" | 번역체/딱딱 |
