# 참고자료: archetype 6종 상세 가이드

> 이 문서는 솔로몬 게임에서 실제로 사용 중인 archetype별 LLM 프롬프트와 끼어들기 패턴을 정리한 것이다.
> 자유 질문 응답 톤 설계 시 이 톤의 **연장선상**에서 설계해야 캐릭터 일관성이 유지된다.

## 1. LLM 심문 프롬프트에서의 archetype 가이드 (blueprintPromptBuilderV2.ts)

### avoidant (회피형)
```
화자는 핵심을 에둘러 말한다.
"사실은~"으로 시작하는 고백을 절대 하지 않고, 절차·과정으로 화제를 돌린다.
감정 표현은 최소, 시선을 피하듯 말한다.
```

### confrontational (대결형)
```
화자는 직접적이고 날카롭다.
상대의 허점을 먼저 지적하고, "오히려 ~가 문제"라는 반격 구조를 즐긴다.
자기 행동은 합리화하면서 상대를 공격한다.
```

### victim_cosplay (피해자 연기)
```
화자는 자기가 가장 큰 피해자라는 서사를 구축한다.
"저는 정말~" "제가 얼마나~" 등 자기중심적 감정 표현이 많다.
상대의 가해 프레임을 강화하려 한다.
```

### cold_logic (냉철 논리)
```
화자는 감정을 철저히 배제하고 사실만 나열한다.
조건부 표현("~라면", "~한 경우에"), 수치, 시간순 서술을 선호한다.
감정 질문에도 사실로 돌려 답한다.
```

### affect_flattening (감정 평탄화)
```
화자는 의도적으로 감정을 눌러 말한다.
짧은 문장, 최소한의 표현, "네", "그렇습니다" 수준의 단답.
진심이 드러나는 것을 꺼린다.
```

### premature_summary (성급한 정리)
```
화자는 대화를 빨리 끝내고 싶어 한다.
"결국~", "핵심은~", "정리하면~" 같은 마무리 표현을 자주 쓴다.
상대의 논점을 축소하고 자기 결론으로 넘어가려 한다.
```

## 2. 끼어들기(interjection) 시스템에서의 실제 대사 예시

### avoidant
- "잠깐만요, 그 부분은 제가 알고 있는 맥락과 좀 다릅니다."
- "재판관님, 그 자료를 그렇게만 보시면 오해가 생길 수 있습니다."
- 고감정: "아니, 잠깐만요! 그건 정말 아닙니다!"

### confrontational
- "아니, 그건 틀렸습니다."
- "재판관님, 상대방이 핵심을 빼놓고 있습니다."
- 고감정: "그만하세요! 거짓말하지 마세요!"

### victim_cosplay
- "재판관님, 저도 할 말이 있습니다..."
- "왜 제 사정은 안 들어보시는 겁니까."
- 고감정: "왜 저만 이러는 겁니까... 저도 사람입니다!"

### cold_logic
- "한 가지 정정하겠습니다."
- "재판관님, 사실관계에 오류가 있습니다."
- 고감정: "더 이상 듣고만 있을 수 없습니다. 명백한 허위입니다."

### affect_flattening
- "그 부분은 다릅니다."
- "사실과 다릅니다. 말씀드리겠습니다."
- (고감정 변형이 거의 없음 — 이것이 이 archetype의 특징)

### premature_summary
- "잠깐, 그건 본 건에서 핵심이 아닙니다."
- "상대방이 핵심을 비켜가고 있습니다. 진짜 문제는 따로 있습니다."
- 고감정: "지금 그게 중요한 게 아니잖습니까!"

## 3. 심문 프롬프트의 좋은/나쁜 대사 예시 (blueprintPromptBuilderV2.ts)

### 좋은 대사 특징
- 구체적 디테일 1개 + 감정 색깔 + 캐릭터 말버릇
- 짧고 날카로운 문장

### 나쁜 대사 특징
- "여러 가지 상황이 얽혀서~" (추상적)
- "~된 것으로 생각됩니다" (보고서 톤)
- "해당 건에 대해서는~" (번역체)

### 실제 좋은 예시 (lieState별)
```
S1(동요): "적지 않은 돈이 움직인 건 맞습니다. 하지만 그 사이에서 생긴 오해가 있어요."
S2(인정): "집안에 급한 일이 생겨서 급히 처리한 건 맞습니다."
S3(전가): "제가 늦었습니다. 그렇다고 상대방이 한 것까지 제 탓은 아닙니다."
S4(호소): "처가 쪽 돈 앞에서 제가 뭘 할 수 있었겠습니까."
S5(인정): "280만원은 오미숙 장모님 돌봄비로 보낸 겁니다. 그게 사실입니다."
```
