## 1. 프로젝트 배경

### 솔로몬 게임이란
한국 민사 분쟁 재판 시뮬레이션 게임. 플레이어가 **재판관**이 되어 양측 NPC(당사자)를 심문한다.

### DossierCard(증거 카드)란
플레이어가 수집한 증거를 묶어 NPC에게 **결정적 질문**을 던지는 시스템.
각 카드에는 3개의 질문이 있고, NPC의 lieState(거짓말 상태)가 올라갈수록 더 날카로운 질문이 해금된다.

### 질문의 화자
**재판관**이 NPC에게 던지는 질문이다. 따라서:
- 재판관 격식체 필수 (~십시오, ~입니까)
- NPC를 "이름 + 씨"로 호칭
- **"당신"** 사용 금지 — 한국 법정에서 재판관은 "당신"을 쓰지 않음
- 유죄/위반을 전제하는 표현 금지
- "~아닌가요" (해요체) 금지 → "~아닙니까" (격식체)

## 2. 교정 대상

3개 사건에서 발견된 **개별 문제 6건**. 해당 질문만 교정하고, 같은 사건의 다른 질문은 건드리지 마세요.

### 사건 정보

**spouse-01 (부부 재산 분쟁):**
- 한지석 (partyA, 남편), 오세린 (partyB, 아내)
- 정우성 (제3자, 중간 경유 계좌)

**workplace-01 (직장 보고서 분쟁):**
- 윤태성 (partyA, 팀장), 박서윤 (partyB, 대리)

**partnership-01 (공동대표 자금 분쟁):**
- 이서준 (partyA, 공동대표), 윤해나 (partyB, 공동대표)

---

## 3. 교정 규칙

### 호칭
- "당신" / "당신이" / "당신도" → 삭제하고 주어 재구성, 또는 "이름 + 씨"
- "해나" (이름만) → "윤해나 씨" 또는 "상대방"

### 종결어미
- "~아닌가요" → "~아닙니까" (해요체→격식체)
- "~해 보십시오" → 허용 (격식체)

### 유죄 전제 금지
- "선행 위반" (위반을 사실로 전제) → "앞서 있었던 일"
- "가담한 것" (가담을 전제) → "관여가 있었는지"
- "심리" (판사가 심리분석을 요구) → "경위" 또는 "판단 과정"

### 재판관 시점 유지
- "제 아내의" (당사자 시점) → "상대방의" 또는 "오세린 씨의"

## 4. 품질 기준

### 좋은 예시 (같은 게임의 다른 사건)
```
"정한결 씨, 관리사무소를 거치지 않고 개인 휴대폰을 세운 이유가 무엇입니까?"
"이서준 씨, 9월 3일 1,800만원 이체를 공동승인 없이 실행한 단말과 승인 방식이 본인 쪽에서 나온 것이 맞습니까."
```

### 금지 표현
| 금지 | 이유 |
|------|------|
| "당신" 전체 | 재판관 어법 위반 |
| "~아닌가요" | 해요체 |
| "가담", "은폐", "조작" (단정형) | 유죄 전제 |
| "제 아내" / "제 남편" | 당사자 시점 |
| "심리" (심리분석 요구) | 재판관 역할 일탈 |
| "선행 위반" (사실 전제) | 스포일러 |
