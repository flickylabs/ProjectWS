# 교정 대상 원본 텍스트 (3개 사건, 6건)

---

## 1. spouse-01 — 1건

### dossier-3.a.q1
- **대상:** 한지석 (partyA)
- **attackVector:** legality
- **현재 텍스트:**
  > "정우성 씨 계좌와 월세 수납증이 모두 원본이라면, **제 아내의** 150만원 자체는 더는 부정하지 않으시겠습니까."

- **문제:** 재판관이 **"제 아내의"**라고 당사자 시점을 채택. 재판관은 제3자 시점이어야 함.
- **교정 방향:** "제 아내의" → "상대방의" 또는 "오세린 씨의". 문장 전체를 재판관 시점으로 재구성.

## 2. workplace-01 — 2건

### dossier-2.b.q3
- **대상:** 박서윤 (partyB)
- **attackVector:** identity
- **현재 텍스트:**
  > "귀하 이름이 찍힌 PDF를 처음 봤을 때 왜 분 단위 시간표를 그렇게 빠르게 내밀었는지, **그 심리까지** 설명해 보십시오."

- **문제:** **"그 심리까지"** — 재판관이 심리분석을 요구하는 것은 역할 일탈. 재판관은 행동의 경위/이유를 묻지, 내면 심리를 묻지 않음.
- **교정 방향:** "그 심리까지" → "그 경위를" 또는 "그렇게 판단한 이유를". 재판관은 관찰 가능한 행동과 판단 과정을 물어야 함.

### dossier-3.a.q2
- **대상:** 윤태성 (partyA)
- **attackVector:** legality
- **현재 텍스트:**
  > "e-5의 소문 확산을 앞세워도, 귀하가 먼저 단독 명의 제출로 규칙을 흔든 사실이 사라지지는 않습니다. **그 선행 위반을 가리기 위해** 상대의 유포를 더 크게 부각한 것은 아닙니까?"

- **문제:** **"선행 위반"**을 사실로 전제 (스포일러). **"가리기 위해"**는 의도를 단정. 재판관은 아직 확정되지 않은 사실을 전제할 수 없음.
- **교정 방향:** "선행 위반" → "앞서 있었던 일". "가리기 위해" → 삭제하고, 질문 형태로 관계를 물어야 함. 예: "앞서 있었던 일과 이번 대응의 관계를 설명하십시오."

## 3. partnership-01 — 3건

### dossier-1.b.q1
- **대상:** 윤해나 (partyB)
- **attackVector:** context
- **현재 텍스트:**
  > "윤해나 씨, **당신도** 전날 예치 한도를 말한 사실은 있지만 9월 3일 실제 송금 버튼과 서면 승인 공백의 책임선은 서준에게 있다고 보십니까."

- **문제:** **"당신도"** — "당신" 사용.
- **교정 방향:** "당신도" → 삭제하고 문장 재구성. "윤해나 씨께서도" 또는 "윤해나 씨 역시" 등.

### dossier-2.a.q3
- **대상:** 이서준 (partyA)
- **attackVector:** legality
- **현재 텍스트:**
  > "직원 불안을 이유로 컨설팅비 명목을 묵인한 순간 **당신도 임시 회계은폐에 가담한 것 아닌가요**."

- **문제 3가지:**
  1. **"당신도"** — "당신" 사용
  2. **"가담한 것"** — "가담"을 전제 (유죄 전제/스포일러)
  3. **"아닌가요"** — 해요체 (격식체가 아님)
- **교정 방향:**
  - "당신도" → "이서준 씨도" 또는 삭제 후 재구성
  - "가담한 것" → "관여가 있었는지" 또는 "연관이 있는 것은 아닌지"
  - "아닌가요" → "아닙니까"
  - 예: "이서준 씨도 그 표기를 묵인한 것은 아닙니까?"

### dossier-3.a.q3
- **대상:** 이서준 (partyA)
- **attackVector:** identity
- **현재 텍스트:**
  > "해나의 임시 분개 제안이 있었다 해도 **당신이** 그 표기를 멈추지 않은 공동책임은 인정합니까."

- **문제:** **"당신이"** — "당신" 사용. 또한 "해나"는 이름만 쓴 것 (줄임말).
- **교정 방향:**
  - "당신이" → "이서준 씨가" 또는 주어 재구성
  - "해나의" → "윤해나 씨의"
