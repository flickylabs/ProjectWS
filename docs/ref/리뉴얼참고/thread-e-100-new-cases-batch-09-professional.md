# Thread-E 100 New Cases

## 배치 9: 의료/교육 (10건)

### 9-01 주황 팔찌의 13분 — professional (`professional-new-01`)

**훅**: 호흡곤란을 호소하던 환자의 대기 시간이 13분 더 늘어난 뒤 보호자 B의 라이브 폭로 영상이 퍼진다. B는 "응급실 책임간호사 A가 병상 부족을 숨기려고 중증도를 낮췄다"고 하고, A는 "B가 대기 중 외부 흡입기를 몰래 써 증상을 흔들고 영상도 편집했다"고 맞선다.  
파고들수록 A는 실제로 병상 회전 압박 속에서 트리아지 등급을 한 단계 낮췄고, B는 기다리는 동안 지인의 흡입기와 해열제를 투여한 뒤 가장 혼잡했던 장면만 잘라 여론을 만들고 있었다.

**A**: 서미정(43, 응급실 책임간호사) — 병상 부족을 이유로 트리아지 등급을 사후 하향 조정한 것 — 거짓말 전략: `cold_logic`  
**B**: 강도윤(35, 환자 보호자) — 외부 흡입기·해열제 투여 사실을 숨기고 대기 영상을 편집한 것 — 거짓말 전략: `victim_cosplay`

**쟁점** (5개):
- d-1: A의 중증도 하향 조정은 오판이었는가, 의도적 병상 관리였는가
- d-2: B의 외부 약물 투여는 긴급 자구행위였는가, 증상 교란이었는가
- d-3: 바이럴 영상은 현장 현실을 보여줬는가, 특정 장면만 과장했는가
- d-4: 응급실 과밀과 개별 오판 중 무엇을 더 무겁게 볼 것인가
- d-5: 생명 안전과 디지털 여론전 중 어느 쪽이 판결을 더 흔드는가

**증거 체인**:
- 초기: e-1 주황 트리아지 팔찌 사진, e-2 B의 라이브 폭로 영상, e-3 초진 활력징후 기록지
- 1차 해금: e-4 EMR 트리아지 수정 로그와 수정 시각표 (requires e-1, S1+)
- 2차 해금: e-5 대기실 CCTV와 지인 흡입기 전달 메신저 캡처 (requires e-2, S2+)
- 3차 해금: e-6 병상 부족 단체채팅과 구급차 전원 지시 메모 (requires e-3, S2+)
- 최종: e-7 스마트워치 산소포화도 기록과 재평가 호출 시각 비교표 (requires e-4+e-5, S3+)
- 핵심 조합:
  - e-1+e-4 → A가 등급을 사후 하향 조정한 사실이 드러난다
  - e-2+e-5 → B가 외부 약물 투여와 대기실 이탈 구간을 영상에서 뺐다는 점이 확인된다
  - e-3+e-6 → 병상 부족 압박이 A의 판단을 왜곡했음을 보여준다

**중반 뒤집기**: 초반엔 A의 중증도 하향이 거의 결정적 잘못처럼 보이지만, e-5와 e-7이 열리면 B도 현장 처치를 흔들고 여론용 영상을 편집했다는 점이 드러난다. 반대로 e-4와 e-6이 열리면 A의 오판이 단순 실수가 아니라 병상 압박 속 선택이었다는 사실도 선명해진다.

**판결 딜레마**: `응급 판단 재량 vs 생명 안전`, `현장 과밀 vs 개별 책임`, `공익 폭로 vs 편집된 여론전`이 충돌한다. B의 영상 편집이 있어도 A의 중증도 조정이 정당화되지는 않는다.

**난이도**: hard

### 9-02 수면검사 84분 — professional (`professional-new-02`)

**훅**: B는 수면클리닉이 `하룻밤 정밀검사` 비용을 받고 실제로는 84분만 측정한 뒤 CPAP 처방까지 밀어붙였다고 폭로한다. A는 "B가 수면제와 에너지음료 복용을 숨기고 센서를 두 번 떼어 결과를 오염시켰다"고 맞선다.  
추적하면 A는 기록 공백을 `artifact resolved`로 처리해 풀검사처럼 청구했고, B는 보험 승인과 휴직 연장을 염두에 두고 약물·카페인 복용 사실을 고의로 누락하고 있었다.

**A**: 유재훈(47, 수면클리닉 원장) — 84분짜리 중단 검사를 정상 풀검사처럼 확정 보고한 것 — 거짓말 전략: `avoidant`  
**B**: 최은서(34, 마케팅 직장인 환자) — 수면제·에너지음료 복용과 센서 탈락 사실을 숨긴 것 — 거짓말 전략: `confrontational`

**쟁점** (5개):
- d-1: A의 검사 보고는 의료적 재량인가, 청구 과장인가
- d-2: B의 약물·카페인 누락은 진단 교란 수준인가
- d-3: CPAP 처방 권고는 실제 데이터에 근거했는가
- d-4: 보험 승인과 병원 청구 중 누가 더 구조를 악용했는가
- d-5: 불완전 데이터의 책임을 어디까지 환자에게 돌릴 수 있는가

**증거 체인**:
- 초기: e-1 검사비 영수증, e-2 최종 수면검사 보고서, e-3 B의 휴직 연장 신청 캡처
- 1차 해금: e-4 센서 탈락 로그와 야간 모니터링 공백 구간표 (requires e-1, S1+)
- 2차 해금: e-5 약국 수면제 구매내역과 에너지음료 결제시간표 (requires e-2, S2+)
- 3차 해금: e-6 AI 초안 판독본과 청구 코드 입력 이력 (requires e-3, S2+)
- 최종: e-7 보험 사전승인 문의 기록과 야간 재부착 간호 메모 (requires e-4+e-5, S3+)
- 핵심 조합:
  - e-1+e-4 → 실제 측정이 중간에 끊긴 불완전 검사였음이 드러난다
  - e-2+e-5 → B가 진단에 영향을 줄 수 있는 복용 사실을 숨겼다는 점이 확인된다
  - e-3+e-6 → A가 불완전 데이터를 정식 청구용으로 확정했다는 점이 열린다

**중반 뒤집기**: 초반엔 A의 청구 과장이 핵심처럼 보이지만, e-5와 e-7이 열리면 B 역시 진단과 보험 결과를 유리하게 만들기 위해 정보를 누락했다는 점이 나온다. 그래도 e-4와 e-6이 열리는 순간 A의 검사 보고 신뢰는 크게 흔들린다.

**판결 딜레마**: `의료 재량 vs 청구 과장`, `정확한 문진 vs 환자 누락`, `보험 승인 vs 진단 무결성`이 충돌한다. B의 누락이 있어도 A가 불완전 검사를 풀검사처럼 청구한 책임은 별도다.

**난이도**: hard

### 9-03 숙제앱의 별 다섯 — professional (`professional-new-03`)

**훅**: 언어치료 종결 직후, B는 `홈 트레이닝 별점 5개`가 전부 자동으로 찍혀 있었다며 환불을 요구한다. A는 "B가 숙제 영상을 조카 영상으로 대체하고 타임스탬프도 바꿨다"고 맞선다.  
파고들수록 A는 지자체 바우처 출석률을 맞추려 숙제앱 체크를 일괄 완료 처리했고, B는 실제로 아이 대신 발음이 더 좋은 사촌 영상과 지난달 영상을 섞어 제출하고 있었다.

**A**: 남유리(36, 소아 언어치료사) — 숙제앱 완료 체크를 일괄 자동 처리한 것 — 거짓말 전략: `affect_flattening`  
**B**: 박지안(33, 보호자) — 조카 영상과 이전 영상을 섞어 숙제를 제출한 것 — 거짓말 전략: `premature_summary`

**쟁점** (4개):
- d-1: A의 자동 체크는 행정 편의였는가, 바우처 실적 조작이었는가
- d-2: B의 숙제 영상 제출은 단순 실수였는가, 의도적 조작이었는가
- d-3: 종결 판단은 실제 아동의 발달을 반영했는가
- d-4: 행정 조작과 보호자 조작 중 무엇이 더 큰 책임인가

**증거 체인**:
- 초기: e-1 숙제앱 별점 화면, e-2 제출 영상 목록, e-3 종결 평가서
- 1차 해금: e-4 앱 관리자 일괄체크 로그와 바우처 정산표 (requires e-1, S1+)
- 2차 해금: e-5 영상 EXIF 정보와 얼굴 인식 불일치 리포트 (requires e-2, S2+)
- 3차 해금: e-6 아동 녹음 원본과 세션별 진전 메모 (requires e-3, S2+)
- 최종: e-7 보호자와의 `이번 주만 맞춰달라` 메신저와 종결 회의 메모 (requires e-4+e-5, S3+)
- 핵심 조합:
  - e-1+e-4 → A가 숙제앱을 실제 수행과 무관하게 일괄 체크했다는 점이 드러난다
  - e-2+e-5 → B가 다른 아이 영상과 과거 영상을 섞어 제출했다는 사실이 확인된다
  - e-3+e-6 → 종결 판단의 근거 자체가 생각보다 빈약했다는 점이 열린다

**중반 뒤집기**: 초반엔 A의 바우처 실적 맞추기가 더 선명한 문제처럼 보이지만, e-5가 열리면 B도 숙제 데이터를 조작해 진전도를 부풀렸다는 점이 나온다. 그래도 e-4가 열리면 A의 평가 시스템 신뢰는 먼저 깨진다.

**판결 딜레마**: `치료 진정성 vs 행정 편의`, `보호자 협조 의무 vs 전문가 평가 책임`, `바우처 실적 vs 실제 아동 상태`가 충돌한다. B의 영상 조작이 있어도 A의 자동 완료 처리는 정당화되기 어렵다.

**난이도**: easy

### 9-04 회복사진의 세 번째 날 — professional (`professional-new-04`)

**훅**: 코 성형 3일 차 회복사진이 병원 SNS에 `빠른 회복 사례`로 올라간 뒤, B는 부종과 멍을 가린 마케팅 사진이라며 폭로한다. A는 "B가 다른 클리닉에서 필러 터치를 받고도 우리 병원 부작용인 척한다"고 맞선다.  
따져보면 A는 홍보 동의서를 수술 후에 다시 받아 넣고 회복 사진을 리터칭했으며, B는 실제로 다른 클리닉에서 조기 필러 수정과 테이핑을 받고 그 사실을 숨긴 채 전액 환불과 협찬 보상까지 요구하고 있었다.

**A**: 정시온(41, 성형외과 원장) — 수술 후 마케팅 동의서를 소급 처리하고 회복 사진을 보정한 것 — 거짓말 전략: `victim_cosplay`  
**B**: 유수빈(27, 뷰티 인플루언서 환자) — 타 클리닉 필러 터치업과 잘못된 후처치를 숨긴 것 — 거짓말 전략: `cold_logic`

**쟁점** (5개):
- d-1: A의 사진 게시와 동의 확보는 적법했는가
- d-2: B의 회복 악화는 원수술 때문인가, 외부 개입 때문인가
- d-3: 리터칭된 회복 사진은 광고 기만인가
- d-4: B의 환불 요구는 정당한 피해구제인가, 협찬형 압박인가
- d-5: 의료 설명의무와 환자 후관리 의무 중 무엇을 더 무겁게 볼 것인가

**증거 체인**:
- 초기: e-1 병원 SNS 게시 캡처, e-2 마케팅 동의서 PDF, e-3 B의 환불 요구 메일
- 1차 해금: e-4 동의서 메타데이터와 전자서명 시각 비교표 (requires e-2, S1+)
- 2차 해금: e-5 타 클리닉 결제내역과 필러 시술 확인 문자 (requires e-3, S2+)
- 3차 해금: e-6 원본 회복사진 EXIF와 리터칭 레이어 비교본 (requires e-1, S2+)
- 최종: e-7 간호사 후관리 안내톡과 B의 브랜드 협찬 제안서 초안 (requires e-4+e-5, S3+)
- 핵심 조합:
  - e-2+e-4 → A가 마케팅 동의를 수술 후 소급 처리한 사실이 드러난다
  - e-3+e-5 → B가 외부 시술 개입 사실을 숨겼다는 점이 확인된다
  - e-1+e-6 → A가 회복 사진을 광고용으로 유리하게 보정했다는 점이 열린다

**중반 뒤집기**: 초반엔 A의 광고용 사진 보정과 동의 소급이 중심처럼 보이지만, e-5와 e-7이 열리면 B도 피해구제만이 아니라 환불과 협찬을 동시에 노렸다는 점이 드러난다. 반대로 e-4와 e-6이 열리면 A의 마케팅 윤리 위반은 선명하게 남는다.

**판결 딜레마**: `광고 투명성 vs 환자 후관리`, `의료 설명의무 vs 외부 개입 은폐`, `피해구제 vs 협찬형 압박`이 충돌한다. B의 외부 시술 숨김이 있어도 A의 동의 소급과 사진 보정은 별도 책임이다.

**난이도**: hard

### 9-05 평가보고서 27쪽 — professional (`professional-new-05`)

**훅**: B는 아이의 ADHD·학습장애 평가보고서 27쪽이 다른 병원 보고서와 문장까지 비슷하다며 재검을 요구한다. A는 "B가 아이에게 샘플 문항을 미리 풀리고 교사 체크리스트도 손봤다"고 맞선다.  
추적해 보면 A는 실제로 과거 보고서 문단을 재사용하며 추가 패키지 권유를 붙였고, B는 특수교육 지원을 받기 위해 교사 체크리스트를 대신 수정하고 유출 문항으로 아이를 연습시켰다.

**A**: 김려원(39, 아동심리 평가센터장) — 과거 보고서 문단을 재사용하고 패키지 연장을 유도한 것 — 거짓말 전략: `confrontational`  
**B**: 윤민서(41, 초등학생 보호자) — 교사 체크리스트와 사전 훈련 문항을 조작한 것 — 거짓말 전략: `avoidant`

**쟁점** (5개):
- d-1: A의 평가보고서는 개별 평가였는가, 템플릿 재활용이었는가
- d-2: B의 사전 훈련은 검사 무효 수준인가
- d-3: 특수교육 지원 필요성은 실제 상태와 얼마나 일치하는가
- d-4: 전문 평가 신뢰와 보호자 조작 중 무엇이 더 판을 흔드는가
- d-5: 추가 패키지 권유는 치료 연속성인가, 상업화인가

**증거 체인**:
