# Thread S v1 Case Design: Spouse

### [spouse] 공유 캘린더의 월요일

**세션 fit 근거**: 이 사건의 핵심은 부부만이 공유하는 공동재산, 배우자 명의 인증 접근, 집 안의 공유 디바이스, 그리고 아이 전학 결정을 서로에게 숨긴 채 먼저 선점하려 한 배신입니다. 연인이나 동업자 분쟁으로는 비슷하게 보일 수 있어도, 이 정도의 금융 접근권과 사적 침범, 양육 관련 선점은 혼인 관계가 아니면 자연스럽게 성립하지 않습니다.

**훅**:
가정 비상금 4,800만 원이 장모 계좌로 사라졌다. 남편은 아내가 아이를 데리고 나갈 준비를 했다고 주장한다.
그런데 아내는 먼저 자기 휴대폰이 밤마다 남편 계정으로 백업됐고, 자기 명의 공동인증서로 추가 담보대출까지 조회됐다고 맞선다.

**A**: 강정민(41, 세무사 사무실 공동운영자) — 아내 명의 공동인증서로 부부 합산 추가 담보대출을 준비했고, 가족용 태블릿을 이용해 아내 휴대폰 백업을 몰래 붙여 놓았다 — `cold_logic`

**B**: 윤서아(39, 초등 학원 상담실장) — 공동 비상금을 친정 계좌로 분산 이체하고, 딸 전학이 가능한 투룸 가계약과 별거 동선을 미리 짜 두었다 — `victim_cosplay`

**쟁점** (4개):
- d-1: 공동 비상금 4,800만 원 이체의 성격은 보호인가, 선점인가 (핵심 잠금: 3회 분할 이체의 이유는 S2에서 열림, 장모 계좌를 우회 보관처로 쓴 사실은 S4에서 열림, 트리거: `hard_evidence` `e-6`)
- d-2: 남편의 공동인증서 접근은 단순 조회인가, 실행 직전 준비인가 (핵심 잠금: 한도 조회 인정은 S1에서 가능, 전자서명 직전 단계까지 간 사실은 S3에서 열림, 트리거: `hard_evidence` `e-5`)
- d-3: 휴대폰/클라우드 감시는 불안의 표현인가, 이혼 대비 사찰인가 (핵심 잠금: 위치/사진 동기화 인정은 S2에서 가능, 메신저 자동백업 유지 사실은 S4에서 열림, 트리거: `hard_evidence` `e-2`)
- d-4: 아이 전학과 별거 준비를 누가 먼저 시작했는가 (핵심 잠금: 집 문의는 S1에서 가능, 전학상담과 실행 일정 선점은 S3에서 열림, 트리거: `hard_evidence` `e-4` + `e-7`)

**증거 체인** (7개):
- e-1: [bank] 가정 비상금 계좌 거래요약 — 1주일 동안 1,600만 원씩 3회, 장모 명의 계좌로 분할 이체됨 (초기 공개)
- e-2: [device] 가족용 태블릿 동기화 기록 — 아내 휴대폰의 사진, 통화기록 요약, 메신저 알림 미리보기가 남편 Apple ID에 심야 자동 동기화되도록 설정됨 (초기 공개)
- e-3: [log] 공동인증서 접속 로그 — 새벽 1시 14분, 남편 노트북에서 아내 명의 인증서로 부부 합산 담보대출 한도 조회가 이뤄짐 (초기 공개)
- e-4: [chat] 부동산 중개 채팅 — 아내가 딸 학교 도보권 투룸과 즉시 전입 가능 날짜를 문의함 (requires: `e-1`, requiredLieState: d-1 `S1`)
- e-5: [contract] 추가 담보대출 상담서 초안 — 배우자 동의 전자서명 링크 직전 단계까지 저장되어 있고, 사용 목적이 "운영자금"으로 적혀 있음 (requires: `e-3`, requiredLieState: d-2 `S2`)
- e-6: [device] 은행앱 푸시알림 삭제 로그 — 아내가 이체 직후 알림을 지우고, 거래 메모를 "학원비/생활비"로 바꿔 흔적을 흐린 기록 (requires: `e-1`, requiredLieState: d-1 `S2`)
- e-7: [log] 공유 가족 캘린더 복원본 — 같은 월요일 오전에는 남편의 "대출 실행 확인", 오후에는 아내의 "전학 면담" 일정이 각각 생성됐다가 삭제된 흔적이 남아 있음 (requires: `e-4`, `e-5`, `e-6`, requiredLieState: d-4 `S3`)

**중반 뒤집기**: 초반에는 아내가 공동재산을 빼돌리고 아이까지 데려나가려 한 사람처럼 보입니다. 그러나 `e-3`와 `e-5`가 열리면 남편이 먼저 배우자 명의 인증서와 집 안 공유 디바이스를 무기화해 추가 대출과 사찰을 준비했다는 사실이 드러납니다. 이후 `e-6`과 `e-7`까지 가면, 아내 역시 단순 방어가 아니라 친정 계좌 은닉과 전학 선점을 통해 먼저 판을 가져가려 했다는 점이 밝혀져, 피해자 프레임이 다시 무너집니다.

**판결 딜레마**: 먼저 사생활과 금융 동의를 침범한 쪽을 더 무겁게 볼지, 아니면 실제로 공동재산을 외부 계좌로 빼돌리고 아이 문제를 단독 선점한 쪽을 더 무겁게 볼지 갈립니다. 절차 위반의 시작점은 남편 쪽이 더 선명하지만, 실행된 생활 파괴의 크기는 아내 쪽이 더 직접적입니다.

**재판관 성향 분화**:
- 논리형: A가 더 나쁘다. 실행 직전까지 간 무단 담보대출 준비와 배우자 디바이스 백업은 동의 없는 권한 침범이 명확하고, 부부 신뢰를 먼저 제도적으로 무너뜨린 행위다.
- 직관형: B가 더 나쁘다. 두려움을 이유로 들더라도 실제로 돈을 친정 계좌로 숨기고 아이 전학을 선점한 순간, 관계를 되돌릴 가능성보다 먼저 상대를 배제하는 선택을 실행했다.
