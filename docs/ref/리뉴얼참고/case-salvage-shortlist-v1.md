# Existing Case Salvage Shortlist v1

## 목적
- 기존 84개 원안 중 `완전 폐기`보다 `고도화 가치가 높은 골격`을 먼저 추리기 위한 1차 메모다.
- 기준은 아래 4가지다.
  - 표면 훅이 빠르게 서는가
  - 양측 공방이 자연스럽게 서는가
  - 증거 해석과 반전 여지가 있는가
  - `재판관 성향 분화`가 살아날 여지가 있는가

## 우선 원칙
- `유지`: 큰 방향이 이미 맞고 밀도만 올리면 되는 케이스
- `고도화`: 골격은 좋지만 더 세게 뒤틀어야 하는 케이스
- `재설계`: 소재만 남기고 쟁점 구조를 다시 짜야 하는 케이스

## 카테고리별 1차 후보

### spouse
- `spouse-05` — `고도화`
  - 잠긴 서재, 녹취, 무단 출입, 업무 보안이 겹쳐 있어 `통제 vs 침범` 축이 강하다.
- `spouse-06` — `고도화`
  - 공개 저격과 불완전한 단서 관리가 겹쳐 있어 `루머/프레임 전쟁`으로 키우기 좋다.
- `spouse-08` — `고도화`
  - 공유 클라우드, 법무사, 과거 대출 문서가 있어 `사생활 침해 + 과거 상처 무기화`가 가능하다.
- `spouse-09` — `고도화`
  - 병원 주차기록, 여성 이름 수취인, 부채 전력이 섞여 있어 `금전 은닉 vs 과잉 의심` 구조가 좋다.
- `spouse-11` — `유지`
  - 공사중지 요청, 제습기 로그, 보험금 공모는 이미 `빌런 vs 빌런` 잠재력이 강하다.
- `spouse-12` — `유지`
  - 오래된 소문, 위조 증거, 현재 허위 유포가 겹쳐 있어 `과거 잘못 vs 현재 조작` 판결이 살아난다.

### friend
- `friend-02` — `유지`
  - 비밀 유출, 무단 열람, 공개 저격, 제3자 조작이 겹쳐 있어 구조가 강하다.
- `friend-03` — `유지`
  - 피해자 코스프레 DM, 선공성 명예훼손, 고의적 증거 위조가 있어 `빌런 vs 빌런` 대표감이다.
- `friend-06` — `고도화`
  - 채널 권한, 공개 비난, 제3자 요청 범위 초과를 `인플루언서/창작자 분쟁` 쪽으로 키우기 좋다.
- `friend-07` — `고도화`
  - 비밀 연애, 편집 캡처, 운영 권력 문제가 있어 `동호회/커뮤니티 권력전`으로 발전 가능하다.
- `friend-09` — `유지`
  - 익명계정, 후기 편집, 경쟁자 몰이 구조가 명확해 자극성과 판결 회색지대를 같이 살릴 수 있다.
- `friend-11` — `고도화`
  - 공유 계정과 개인정보, 관리자 권한 변경 문제는 `사생활 침해 vs 접근 차단` 축이 좋다.

### tenant
- `tenant-02` — `유지`
  - 곰팡이, 무단 출입, 타임스탬프 조작이 있어 `하자 대응 vs 사생활 침범` 구조가 매우 강하다.
- `tenant-03` — `고도화`
  - 추가 근저당, 중개사 개입, 개인 계좌 송금은 `전세사기형`으로 키우기 쉽다.
- `tenant-08` — `유지`
  - 관리비 원본 고지서 미제출과 도장 합성은 명확한 악행이라 `실사용 vs 조작` 판별이 선명하다.
- `tenant-09` — `유지`
  - 최종 계약서, 실제 수령액, 공모 구조가 있어 `구조 사기 + 확인 책임` 딜레마가 강하다.
- `tenant-12` — `고도화`
  - 관리인 발 소문, 늦은 정정, 평판 훼손은 `이웃/건물 커뮤니티 루머전`으로 확장 가능하다.

### workplace
- `workplace-02` — `유지`
  - 다운로드 로그, 외부 메시지, 제3자 유출, 입막음 지시가 있어 내부고발형으로 키우기 좋다.
- `workplace-05` — `유지`
  - 아이디어 탈취처럼 보이지만 피해자 위치 선점 문제까지 있어 아주 강하다.
- `workplace-07` — `유지`
  - 익명 게시판, 선행 보복, 편집 증거 유포가 겹쳐 `제보 vs 조작` 축이 선명하다.
- `workplace-08` — `고도화`
  - 법인카드, 비공식 합의, 오프북 정산은 `회계 관행 vs 책임전가` 쪽으로 밀 수 있다.
- `workplace-11` — `유지`
  - 발명자 삭제, 브로커 전달, 인사개입이 겹쳐 있어 `직장 대표 강사건` 후보로 적합하다.
- `workplace-12` — `유지`
  - 재택 모니터링과 편집 증거는 `성과 관리 vs 사생활 침해` 축이 분명하다.

### partnership
- `partnership-04` — `유지`
  - 현재 증거 조작과 과거 실제 배신이 겹쳐 있어 `지금의 위조 vs 과거의 진짜 잘못`이 갈린다.
- `partnership-05` — `고도화`
  - 자문사 계좌와 심야 이체는 `공금 운용 vs 유용` 회색지대로 강화 가능하다.
- `partnership-07` — `유지`
  - 외부 운영사 전달, 누설자 몰이, 명예훼손 구조가 강하다.
- `partnership-09` — `유지`
  - 지원금 공로 다툼처럼 보이다가 공동 허위 제출이 드러나는 구조가 매우 좋다.
- `partnership-11` — `유지`
  - 새 지점 선점과 별도 법인 준비가 맞물려 `절차 위반 vs 선행 배신` 구조가 강하다.

### neighbor
- `neighbor-02` — `고도화`
  - CCTV 유포, 악취 민원, 배후 조작을 `커뮤니티 여론전` 쪽으로 발전시키기 좋다.
- `neighbor-03` — `고도화`
  - 반려견 공격 오해와 편집 영상은 감정 몰입과 증거 해석이 잘 붙는다.
- `neighbor-05` — `유지`
  - 택배 절도 프레임, CCTV 유포, 역소문 전파가 있어 작은 사건인데도 강하다.
- `neighbor-08` — `유지`
  - 이미 `스토킹 낙인 vs 과잉 방어` 구조가 강해서, headline급으로도 키울 수 있다.

## 현재 판단
- `유지` 후보는 큰 구조를 살리고 디테일과 자극도를 올리는 편이 맞다.
- `고도화` 후보는 표면 훅을 더 세게 만들고, 양측 숨긴 사실을 더 분명히 추가해야 한다.
- `재설계` 대상은 아직 전수 표시하지 않았지만, 지나치게 순한 금전 정산형/오해 해소형 케이스들이다.
