## 보고 형식

**케이스별로 5개 Level 각각 PASS/WARN/FAIL + 이슈 원문.**

특히 **Level 4는 카테고리별(4-A~4-H)로 분리 보고**해 주세요:

```
## [caseId]

### Level 1: PASS
(세부 생략 가능 — 전부 PASS면)

### Level 2: PASS
(세부 생략 가능)

### Level 3: PASS
(세부 생략 가능)

### Level 4: WARN
- 4-A: PASS
- 4-B: WARN — 4-B3 재판관이 "남편 분은~" 호칭 사용 (원문: "...")
- 4-C: PASS
- 4-D: WARN — 4-D1 partial인데 5문장 (원문: "...")
- 4-E: PASS
- 4-F: WARN — 4-F1 "정태성는" 조사 오류
- 4-G: PASS
- 4-H: PASS

### Level 5: PASS

### 종합: 조건부 PASS (Level 4 WARN 3건)
```

**PASS인 Level은 세부 생략 가능. WARN/FAIL인 Level은 반드시 해당 항목 번호 + 원문 포함.**
