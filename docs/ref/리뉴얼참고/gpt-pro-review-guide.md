# 솔로몬 법정 — 사건 기획 검토 가이드 (GPT Pro 5.4용)

---

## 이 문서의 목적

당신은 게임 **솔로몬 법정**의 사건 기획 100건을 검토하는 역할입니다.
핵심 임무: **재미있는 사건은 살리고 발전시키고, 재미없는 사건은 과감히 버려 주세요.**

---

## 1. 게임은 무엇인가

### 한 줄
**AI 캐릭터 둘이 갈등하고, 플레이어(재판관)가 심문·증거·중재로 진실을 밝히는 추리 재판 게임.**

### 플레이 루프
```
사건 소개 → 양측 진술 듣기 → ★심문(질문 선택) → ★증거 제시/조사 → ★재심문 → 중재 → 판결 → 결과
```

★ 표시된 3단계가 핵심 플레이입니다. 플레이어가 **질문을 고르고, 증거를 들이밀고, 거짓말을 깨뜨려서** 진실에 도달합니다.

### 핵심 쾌감
- **"이 사람, 거짓말하고 있었구나"** — 추궁으로 거짓말이 무너지는 순간
- **"이 증거가 이걸 증명하는 거였어?"** — 증거의 의미를 발견하는 순간
- **"피해자라고 생각했는데 이 사람도..."** — 판이 뒤집히는 순간
- **"둘 다 잘못했는데 어떻게 나누지..."** — 판결이 고민되는 순간

---

## 2. 재미의 기준 — 이것으로 판단해 주세요

### 재미있는 사건의 5가지 조건

각 사건을 아래 5개 기준으로 평가해 주세요. **5개 중 4개 이상 충족하면 합격, 3개면 조건부, 2개 이하면 탈락입니다.**

#### ① 양측 모두 나쁘다 (대칭적 갈등)
> "한쪽이 명백히 나쁜 사건은 재판할 이유가 없다"

- A도 숨긴 게 있고, B도 숨긴 게 있는가?
- 둘 다 자기에게 유리하게 프레이밍하고 있는가?
- 플레이어가 "A가 나쁘다"고 생각했다가 "B도 나쁘네"로 바뀌는 순간이 있는가?
- **최악**: 한쪽이 명백한 악당. 재판 필요 없음.

#### ② 증거가 이야기를 만든다 (증거 체인)
> "증거를 하나씩 열 때마다 사건의 모습이 달라져야 한다"

- 초기 증거 3개만으로는 사건의 전모를 알 수 없는가?
- 새 증거가 열릴 때마다 **"아, 이런 거였어?"**라는 반응이 나오는가?
- 증거 2개를 조합하면 개별로는 안 보이던 사실이 드러나는가?
- **최악**: 증거를 안 봐도 질문만으로 전부 알 수 있음. 증거 시스템이 장식.

#### ③ 중반에 판이 뒤집힌다 (반전)
> "처음 30%는 한쪽이 거의 확실히 나빠 보여야 하고, 중반에 완전히 뒤집혀야 한다"

- 초반에 플레이어가 "A가 나쁘다"고 확신하게 유도하는가?
- 중반(새 증거 해금 or 상대측 추궁)에서 그 확신이 깨지는가?
- 뒤집힌 후에도 **처음 판단이 완전히 틀린 건 아닌** 구조인가? (양쪽 다 나쁨)
- **최악**: 처음부터 끝까지 같은 구도. 뒤집기 없음.

#### ④ 판결이 고통스럽다 (딜레마)
> "정답이 없어야 한다. 어떤 판결을 내려도 뭔가 찜찜해야 한다"

- 논리적 재판관과 감성적 재판관이 **다른 판결**을 내릴 수 있는가?
- "현재의 잘못 vs 과거의 잘못", "절차 vs 실체", "용서 vs 제재" 같은 축이 교차하는가?
- 플레이어가 판결 후 **"다른 선택을 했으면 어땠을까"**라고 생각하게 되는가?
- **최악**: 명백한 정답이 있음. 고민 없이 판결 가능.

#### ⑤ 현실에서 일어날 법하다 (공감)
> "나도 이런 상황에 처할 수 있겠다"

- 한국 사회에서 실제로 벌어질 수 있는 소재인가?
- 등장인물의 동기가 이해 가능한가? (악당이 아니라 사정이 있는 사람)
- 플레이어가 자기 경험에 비추어 감정 이입이 되는가?
- **최악**: 비현실적 설정, 공감 불가능한 동기.

## 3. 사건의 구조 — 어떤 부품이 있어야 하는가

### 필수 부품

| 부품 | 역할 | 없으면 |
|------|------|--------|
| **훅** | 첫 2줄에서 "이건 뭐지?" 끌림 | 플레이어가 사건을 선택하지 않음 |
| **양측 숨긴 사실** | A도 B도 진실을 말하지 않음 | 한쪽만 추궁하면 끝 |
| **증거 체인** | 초기 3개 → 조건부 해금 4개 | 탐색의 재미 없음 |
| **증거 조합** | 2개를 합치면 새 사실 | "아하!" 순간 없음 |
| **중반 뒤집기** | 50% 시점에서 판이 바뀜 | 지루함 |
| **판결 딜레마** | 어떤 판결도 100% 옳지 않음 | 게임 끝나도 여운 없음 |

### 증거 구조 상세

```
초기 제공 3개: 플레이어가 처음 보는 증거. 표면적 사실만 알 수 있음.
1차 해금 (S1+): 심문으로 거짓말을 S1까지 깨면 열림. "아, 이런 것도 있었어?"
2차 해금 (S2+): 더 깊이 추궁하면 열림. "이게 진짜 핵심이었네"
최종 해금: 복수 증거를 제시해야 열림. 결정적 증거.
증거 조합 3개: 두 증거를 합치면 개별로는 안 보이던 사실이 드러남.
```

### 거짓말 상태 (LieState)

```
S0: 완전 부정 → S1: 일부 인정 → S2: 핑계 → S3: 책임 전가 → S4: 감정 폭발 → S5: 자백
```

각 단계에서 NPC가 공개하는 정보 수준이 다릅니다:
- S0-S1: 모호한 표현만 ("그 돈", "그 사람")
- S2: 범위형 ("200만원대", "김 씨")
- S3+: 구체적 (실명, 정확한 금액)
- S5: 전부 공개

### 심문 3종

| 유형 | 톤 | 효과적인 상황 |
|------|---|-------------|
| 사실 추궁 | 단호, 논리적 | 증거가 있을 때, 모순이 발견되었을 때 |
| 동기 탐색 | 탐색적, "왜?" | 숨긴 동기를 파악할 때, 행동의 이유를 물을 때 |
| 공감 접근 | 부드러움 | 감정적 NPC, 신뢰를 쌓을 때, 자발적 자백 유도 |

**좋은 사건은 "이 NPC에게는 이 접근이 먹힌다"는 최적 경로가 있어야 합니다.**

### 재판관 성향 (판결이 갈리는 축)

| 축 | 이쪽 | 저쪽 |
|---|------|------|
| 현재 vs 과거 | 지금 잘못이 더 크다 | 과거 원인이 더 근본적 |
| 절차 vs 실체 | 절차 위반이 잘못 | 실질 피해가 잘못 |
| 의도 vs 결과 | 나쁜 의도가 핵심 | 나쁜 결과가 핵심 |
| 보호 vs 통제 | 보호하려 한 것 | 통제하려 한 것 |
| 투명성 vs 사생활 | 알 권리 | 숨길 권리 |
| 용서 vs 제재 | 사정 참작 | 선례 방지 |

**좋은 사건은 위 축 2~3개가 동시에 교차합니다.**

## 4. 당신에게 요청하는 것

### 100건 각각에 대해

1. **재미 점수** (1~10): 위 5가지 기준의 종합. 7점 이상이면 합격.
2. **강점**: 이 사건이 재미있는 이유 (1줄)
3. **약점**: 이 사건의 가장 큰 문제 (1줄)
4. **판정**: `KEEP` / `DEVELOP` / `DROP`
   - `KEEP`: 현재 기획 그대로 생성기에 넣어도 됨
   - `DEVELOP`: 재미 잠재력은 있지만, 특정 부분을 고쳐야 함 (구체적 제안 필수)
   - `DROP`: 가망 없음. 수정해도 재미 기준을 못 넘을 것 (이유 필수)

### DEVELOP 판정 시 구체적 제안

"훅이 약하다"가 아니라, **"훅을 이렇게 바꾸면 된다"**를 써 주세요.
예:
```
DEVELOP — 훅 변경 제안: "리모델링 누수"로 시작하지 말고, 
"보험회사에서 '부부가 같은 날 같은 사고를 신고했다'는 연락이 오면서 시작"으로 바꾸면
초반부터 공모 의심을 심을 수 있음
```

### 추가로

- **빠진 카테고리나 소재 영역**이 있으면 제안해 주세요
- **기존 사건과 너무 비슷한** 것이 있으면 지적해 주세요
- **100건 전체를 본 후**, 상위 10건 / 하위 10건을 별도로 뽑아 주세요

## 5. 나쁜 사건의 패턴 — 이것이 보이면 DROP

| 패턴 | 왜 나쁜가 |
|------|----------|
| **한쪽 명백 악당** | 재판 필요 없음, 판결 고민 없음 |
| **오해 해소형** | 진실 밝혀지면 갈등 소멸, 게임 끝남 |
| **제3자가 진범** | A vs B 대결이 약해짐, 당사자 공방 없음 |
| **금액만 다투는 정산** | 증거 해석전이 없음, 숫자 맞추기 |
| **범죄 수사물** | 이 게임은 민사 분쟁 재판. 살인/절도 X |
| **감정만 있고 증거 없음** | 심문만 하면 끝, 증거 시스템 장식 |
| **stakes가 너무 낮음** | 6만원 수리비 분쟁 — 긴장감 없음 |
| **설정이 비현실적** | 공감 불가, 한국 사회와 동떨어짐 |

---

## 6. 좋은 사건의 패턴 — 이것이 보이면 가산점

| 패턴 | 왜 좋은가 |
|------|----------|
| **양측 공모 후 배신** | "같이 했는데 누가 더 나빠?" — 가장 강력한 딜레마 |
| **같은 증거 양쪽 해석** | "이 CCTV는 A가 나쁜 건가 B가 편집한 건가?" |
| **피해자가 가해자로** | 동정했던 사람이 실은 가담자 — 최고의 반전 |
| **디지털 증거 핵심** | 채팅/SNS/앱 기록 — 현대적이고 검증 가능한 증거 |
| **시간차 진실** | 과거의 잘못과 현재의 잘못이 다른 축 — 판결이 갈림 |
| **전문가 권한 남용** | 의사/교사/변호사의 위치 이용 — 공감+분노 동시 |
