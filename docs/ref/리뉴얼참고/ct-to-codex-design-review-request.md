| 축 | 음(-) | 양(+) |

## 2. 게임 디자인 확장 (방향 합의 완료)

### 배경
현재 게임의 핵심 문제: 플레이어 에이전시 부족 — 내부 상태 변화(거짓말 전이, 방어 약화)가 플레이어에게 새로운 행동 선택지로 드러나지 않음.

### 합의된 5대 개선 방향

#### (1) 증거 시스템 재설계
- 실행 가능 증거 카드를 6→3~4개로 축소
- 증거별 타겟 특화 추궁 질문 추가
- 방어 붕괴를 시각적으로 표시

#### (2) 심문 3종 효과 분리
- 3 액션 유지하되 게임 효과를 명확히 차별화
- 모순 감지 / 감정 누설 / 신뢰 구축의 실질적 차이
- 시각 효과 아이콘으로 결과 피드백

#### (3) 돌파 순간 연출
- 상태 전이마다 시각 피드백:
  - S1→S2: 균열 (crack)
  - S2→S3: 궁지 (cornered)
  - S3→S4: 개방 (opening)
  - S4→S5: 자백 (confession)
- 버튼 추천 변경 (다음에 뭘 하면 좋을지)

#### (4) 턴 기반 이벤트 부활
- 4종 조건부 이벤트: 모순 지적, 끼어들기, 감정 폭발, 숨겨진 쟁점 발현
- 플레이어 선택지 제공 (장식이 아닌 실질적 분기)

#### (5) 상태 기반 정보 해금
- 거짓말 상태 전이 → 새로운 정보 레이어 해금
- 쟁점별 "freshness budget"으로 무한 반복 방지

### 현재 구현 상태
- Phase 2 Wave 1~3에서 (3), (4)의 일부 구현 완료 (돌파 연출, 교착 피드백, 심문 효과 칩)
- (1), (2), (5)는 미구현

### 리뷰 요청
7. Phase 2에서 구현된 부분의 품질/완성도에 대한 의견
8. 미구현 항목 (증거 축소, 심문 효과 분리, freshness budget) 중 가장 임팩트가 큰 것은?
9. PC 버전에서 이 확장들이 모바일과 다르게 구현되어야 하는 점은?
10. "증거 카드 3~4개로 축소"가 정말 맞는가? 현재 6개 유지가 더 나을 수 있는가?
11. 돌파 순간 연출과 PC 프로토타입의 Phase 전환 연출이 충돌하지 않는가?

## 3. Phase 4 설계 — BM/서버/시즌 (컨셉 합의)

### 마스터 플랜 순서 (변경 불가)
Phase 1 품질 잠금 → Phase 2 에이전시+재미 → Phase 3 밸런스 → **Phase 4 BM**

### BM 컨셉

#### 시즌 패스 (배틀패스)
- 가격: 4,900원 / 30일 주기
- 무료/프리미엄 트랙
- 미션 기반 진행 (재판 완료, 특정 칭호 획득 등)
- 보상: 코스메틱, 재판관 스킨, 칭호 프레임

#### 프리미엄 사건 팩
- 시즌 사건 (1,900원)
- 5건 팩 (2,900원)
- 레전더리 사건 (3,900원)

#### 수익 모델 추정
- DAU 1K~10K 기준
- 광고 + 시즌패스 (전환율 5%) + 사건팩 (전환율 2%)
- 예상: 월 82만~828만원

#### 서버 인프라
- 리더보드/시즌 추적 스텁: `leaderboard.ts`, `leaderboard` 타입
- 드리프트 상태 직렬화 (서버 동기화 준비)
- 시즌 날짜 매핑

### 현재 상태
- **의도적으로 후순위**: 코어 재미가 검증되기 전 BM 설계는 스코프 크립 위험
- 코드: 리더보드 스텁만 존재, 실제 서버 없음

### 리뷰 요청
12. Phase 4를 Phase 1~3 이후로 미루는 전략이 맞는가? 동시 병행해야 할 부분은?
13. 시즌 패스 모델이 이 장르(추리/재판)에 적합한가? 다른 BM 모델이 더 나을 수 있는가?
14. 프리미엄 사건 팩의 가격 책정이 적절한가?
15. 리더보드가 이 게임에 의미가 있는가? (경쟁 요소 vs 개인 성찰형 게임)
16. PC(Steam) 출시와 모바일 출시의 BM 차이를 어떻게 가져가야 하는가?
17. 재판관 성향 시스템 + 시즌 패스의 시너지 가능성은?
