## Part 2: 재미 증대

### 2-1. 현재 재미 구조 분석

**강점**:
- 진실을 직접 밝혀내는 탐정 쾌감 (코어 루프)
- S0→S5 점진적 자백의 서사적 만족감
- 84건 다양한 사건 (7카테고리 × 12건)
- 3축 점수로 다양한 플레이 스타일 인정

**약점/개선 기회**:

#### 문제 1: 심문 3종 액션의 체감 차이 부족
- **현상**: fact_pursuit / motive_search / empathy_approach 선택이 결과에 큰 차이를 만들지 못함
- **원인**: 재판관 질문은 다르지만 NPC 응답이 비슷하게 느껴짐
- **개선안**:
  - **A. 시각적 피드백 강화**: 액션 선택 후 "모순 토큰 +1", "누설 미터 +15%" 같은 수치 변화를 즉각 표시
  - **B. 고유 후속 효과**: empathy_approach 성공 시 "신뢰 창구 열림 → 다음 턴 특수 선택지" 제공
  - **C. 콤보 시스템**: fact→fact→motive 연속 시 "압박 콤보" 보너스

#### 문제 2: 중반부 느슨함
- **현상**: S2~S3 구간에서 같은 패턴 반복 (질문→회피→질문→회피)
- **개선안**:
  - **A. 이벤트 트리거 확대**: S2 도달 시 자동 이벤트 (증인 자발적 등장, 새 증거 발견)
  - **B. 시간 압박**: "남은 턴: 8" 표시 → 긴장감 유지
  - **C. NPC 끼어들기 빈도 조절**: S2~S3에서 상대방 끼어들기 확률 ↑

#### 문제 3: 증거 조사의 깊이감 부족
- **현상**: 증거를 클릭하면 정보가 나오지만, "발견의 쾌감"이 약함
- **개선안**:
  - **A. 증거 연결 시각화**: 증거 A + 증거 B를 연결하면 새로운 사실 발견 (이미 조합 시스템 있음, UI 강화)
  - **B. "아하!" 모먼트**: 핵심 증거 발견 시 특별 연출 (화면 효과 + 사운드)
  - **C. 증거 감별의 의미**: 현재 trustworthy/partial/suspicious 감별이 게임에 미치는 영향 강화

#### 문제 4: 판결의 무게감
- **현상**: Phase 7에서 책임 배분 슬라이더만 있어서 단순하게 느껴짐
- **개선안**:
  - **A. 판결 전 요약**: "지금까지 밝혀진 사실" 자동 정리 화면
  - **B. 판결 결과의 서사**: 후일담에 판결 결과가 구체적으로 반영 (이미 구현, 강화 필요)
  - **C. "다른 재판관의 판결"**: 동일 사건에 대한 AI/다른 플레이어의 판결 비교

### 4-1. 현재 BM 구조

```
[무료 경로] 자동충전(1시간) + 광고(5회/일) → 1일 1~2게임 가능
[유료 경로] 상점 (미구현) → 토큰/포인트 직접 구매
```

### 4-2. BM 강화 제안

#### A. 시즌 패스 (배틀패스)

| 등급 | 가격 | 혜택 |
|------|------|------|

- 미션 기반 진행 (일일/주간/시즌)
- 감정 표현 스킨 (MS 이모지 변형)

#### C. 편의 아이템

| 아이템 | 가격 | 효과 |
