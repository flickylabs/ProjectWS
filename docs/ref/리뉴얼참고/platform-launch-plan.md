# 솔로몬 법정 — 출시 계획 1장 요약

---

## 출시 순서

```
[내부 테스트] → [PC Steam EA/정식] → [모바일 소프트런칭] → [모바일 정식]
                 ↑ 1차 출시               1~3개월 후
```

- PC 먼저 출시하여 프리미엄 포지셔닝 확립
- 모바일은 PC 피드백 반영 후 소프트런칭
- EA 시 3~6개월 이하 (텍스트 게임은 "미완성" 인상 위험)

## 기술 구조

```
src/
├── engine/           ← 공유 (48개 룰 엔진, 변경 없음)
├── store/            ← 공유 (Zustand 8슬라이스)
├── types/            ← 공유
├── data/             ← 공유 (84건 사건 데이터)
├── hooks/
│   ├── shared/       ← 게임 로직 훅 (상태/규칙)
│   ├── mobile/       ← 모바일 UI 훅
│   └── pc/           ← PC UI 훅 (드래그, 키보드, 호버)
├── components/
│   ├── common/       ← 공유 (프리미티브, 결과, 설정, 도메인 카드)
│   ├── mobile/       ← 모바일 전용 (기존 16모듈 이동)
│   └── pc/           ← PC 전용 (프로토타입 기반 신규)
├── app/
│   ├── MobileApp.tsx
│   └── PCApp.tsx
└── platform/
    ├── web.ts        ← localStorage, Web Audio, 광고 SDK
    └── steam.ts      ← Steamworks, Cloud Save, Achievements
```

### 빌드
- `vite.config.base.ts` — 공통 설정
- `vite.config.mobile.ts` — 모바일 진입점 (MobileApp.tsx)
- `vite.config.pc.ts` — PC 진입점 (PCApp.tsx)
- `npm run build:mobile` / `npm run build:pc`

### PC 래핑
- 1차: **Electron** (성숙, Steam 연동 안정)
- 향후: Tauri 검토 (경량화)

### PC 입력 격리
- hover/drag/keyboard/right-click → **intent 변환 어댑터**
- `compare_notes`, `focus_evidence`, `open_palette` 등 공통 액션
- 컴포넌트는 intent만 소비, 플랫폼 어댑터가 입력 변환 담당
