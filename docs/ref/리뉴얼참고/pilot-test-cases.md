# spouse-01 파일럿 테스트 케이스 (V2)

## 테스트 범위
- 심문 (사실추궁/동기탐색/공감접근) × 양측 × 복수 쟁점
- 증거 제시 (e-1, e-2, e-3) × 양측
- 상태 진행 (S1→S2→S3→S4→S5)
- 캐릭터 차이 (avoidant vs confrontational)
- 반복 방지 (tell cadence, 표현 다양성)
- 호칭 규칙

## 자동 검증 항목
- 자기모순 ("그런 적 없습니다" + "맞지만")
- 재판관 앞 애칭 ("자기가")
- "9월 20일 기준으로" 패턴
- tell 없이 exact 금액 노출
- "순서" 고착 패턴
- 반말 종결 / 재판관에게 되묻기
- 이름 호칭 ("지석 씨", "세린 씨")

## 최신 테스트 결과 (V2 atom + speechStyle 제거)
- 20턴 (심문 16 + 증거 3 + 스킵 1)
- ✅ 16 PASS | ⚠️ 3 WARNING (모두 exact 금액) | ❌ 0 ERROR
- "9월 20일 기준으로": 0회
- 자기모순: 0회
- 반말/되묻기: 0회
- 상태 진행(S1→S5): 자연스러운 톤 변화 확인

## 미테스트 / 알려진 한계
- 증거 조사 6종 (request_original 등) 미테스트
- 증인 소환 미테스트
- Discovery 트리거 미테스트
- 전체 게임 흐름 (readiness → 조정 → 판결) 미테스트
- exact 금액이 S4/S5 고백 단계에서 LLM 자체 생성 (3/20턴)
- 오세린 b/d-1 V2 atom 미보유 → 증거 e-1 양측 제시 불가
