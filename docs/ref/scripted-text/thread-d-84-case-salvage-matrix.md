# Thread-D 84 Case Salvage Matrix

## 분류 메모
- `keep`: 큰 틀 유지, 디테일만 강화하면 된다.
- `upgrade`: 골격은 좋고 강한 리라이트 가치가 있다.
- `rebuild`: 소재만 살리고 구조는 사실상 다시 짜야 한다.
- `drop`: 유지 가치가 낮고 신규 사건이 더 싸다.

형식:
`caseId | category | classification | one-line verdict | salvageable assets | core weakness | recommended rewrite direction | possible A/B framing | judge-profile divergence potential | priority`
