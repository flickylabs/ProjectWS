# 솔로몬 앱 플로우 설계

> 최종 업데이트: 2026-03-27

---

## 1. 전체 플로우

```
최초 실행 → 인트로 → 로그인 → 홈 → 사건 맵 → 플레이 → 결과
                                 ↕
                            프로필 / 상점 / 리더보드 / 설정
```

## 2. 최초 인트로 (1회만)

```
[화면 1] 로고 + 태그라인
"⚖️ 솔로몬"
"AI 둘의 싸움을 인간의 지혜로 재판하는 게임"

[화면 2] 세계관
"사람들 사이의 갈등 — 누가 옳고 누가 그른지, 답은 단순하지 않습니다."
"당신은 재판관. 양쪽의 말을 듣고, 증거를 조사하고, 현명한 판결을 내려야 합니다."

[화면 3] 핵심 룰
"진실은 하나가 아닙니다. 양쪽 모두 거짓말을 하고 있을 수 있습니다."
"심문하고, 증거를 들이밀고, 숨겨진 진실을 밝혀내세요."

[화면 4] 시작
"당신의 첫 번째 사건이 기다리고 있습니다."
[재판 시작 →]
```

## 5. 사건 맵 (스테이지)

### 맵 구조
```
관계 유형별 챕터 → 챕터 안에 12개 스테이지

챕터 1: 부부 ❤️ (spouse)
  ├── Stage 1: "추석의 비밀 송금" ⭐⭐⭐ 명성 50
  ├── Stage 2: "상담기록 유출" ⭐⭐☆ 명성 35
  ├── Stage 3: "이직 후 가계 공백" ☆☆☆ (미도전)
  ├── ...
  └── Stage 12: "20년 전 소문" 🔒 (Stage 10 클리어 필요)

챕터 2: 이웃 🏠 (neighbor) — 챕터 1의 Stage 3 클리어 시 해금
  ├── Stage 1: "빗물과 자정의 소음" ☆☆☆
  └── ...

챕터 3: 직장 💼 (workplace) — 챕터 2의 Stage 3 클리어 시 해금
...

챕터 7: 친구 🤝 (friend)
```

## 7. 이벤트씬 설계

### 이벤트씬 종류
```
1) 사건 인트로씬 — 사건 시작 전, 배경 설정
2) 핵심 반전씬 — 결정적 증거/진실 발견 시
3) 엔딩씬 — 판결 후 후일담
```

### spouse 사건별 이벤트씬 제안

| 사건 | 인트로씬 | 반전씬 | 엔딩씬 |
|------|---------|--------|--------|
| spouse-01 (추석 비밀 송금) | 추석 직전 부엌에서 장을 보는 두 사람. 남편 폰이 울린다 | 돌봄센터 예약서를 펼치는 순간 — 여자 이름이 간병사였다 | 추석 상 차리며 "그때 왜 말 안 했어" 대화 |
| spouse-02 (상담기록 유출) | 입학식 앞 학교 정문. 다른 부모의 수군거림 | 보조 이메일의 스캔 기록 발견 — 남편이 올린 것 | 아이 손 잡고 등교. "엄마 괜찮아?" |
| spouse-03 (이직 후 공백) | 텅 빈 냉장고 앞에 선 아내. 남편의 사직서 | 양쪽 비밀 계좌가 같은 주에 터진 것 | 함께 가계부를 다시 쓰는 모습 |
| spouse-04 (기념일 약속 파기) | 레스토랑 예약 확인 알림 | 시어머니 주차장에서의 통화 녹음 | 둘만의 늦은 저녁, "다음엔 말해줘" |
| spouse-05 (서재 공간 분쟁) | 좁은 서재에 책상 두 개가 마주보고 있다 | 녹취 파형이 편집된 것 발견 | 벽을 허물고 넓힌 서재에서 각자 작업 |
| spouse-06 (SNS 오해) | 스토리에 올라간 사진 한 장 | 게시 시각과 해명문 시각 6분 차이 | 서로의 SNS 비밀번호를 바꾸는 장면 |
| spouse-07 (육아 기여도) | 아침 등원 준비 — 한쪽만 분주한 장면 | 육아 분담표의 실제 데이터 공개 | 아이가 "오늘은 아빠가 해줬어" |
| spouse-08 (전세갱신 갈등) | 카페 합의 장면 — 불안한 두 얼굴 | 대출 정리서류의 타임라인 불일치 | 새 집 열쇠를 함께 받는 장면 |
| spouse-09 (간병보험금) | 병원 복도에서 마주치는 두 사람 | 보험금 930만원의 실제 흐름도 | 어머니 병실에서 함께 식사 |
| spouse-10 (명절 일정 충돌) | 네비에 서로 다른 목적지가 찍힌 두 폰 | 같은 시각 다른 방향으로 차를 빼는 CCTV | 양가 화상통화로 인사하는 장면 |
| spouse-11 (리모델링 누수) | 천장에서 물이 떨어지는 거실 | 보험 신청서 작성 시각이 누수 전 | 함께 페인트칠하는 벽 |
| spouse-12 (20년 전 소문) | 오래된 메시지 캡처를 손에 든 모습 | 캡처의 2008년 스킨+2024년 템플릿 레이어 | 졸업앨범을 함께 보며 웃는 장면 |

### neighbor 사건별 이벤트씬 제안

| 사건 | 인트로씬 | 반전씬 | 엔딩씬 |
|------|---------|--------|--------|
| neighbor-01 (누수 소음) | 장마 밤, 천장 얼룩을 올려보는 A | 설비 점검서 — 세탁기가 아니라 빗물 역류 | 복도에서 인사하는 두 사람 |
| neighbor-02 (반상회 소문) | 반상회 뒤 택배실 앞에서 마주침 | 편집된 CCTV 캡처 — 동대표가 퍼뜨림 | 동대표 사과문이 게시판에 |
| neighbor-03 (반려견 사고) | 복도에서 두 개가 마주친 순간 | CCTV 위조 — 시간표시 레이어 발각 | 산책길에서 함께 걷는 두 개 |
| neighbor-04 (주차 분쟁) | 아침 출근길 — 막힌 차 앞에 선 A | 흠집 사진이 몇 달 전 옛 사진 | 주차면에 작은 화분을 함께 놓는 장면 |
| neighbor-05 (택배 분실) | 현관 앞 빈 자리 — 택배가 없다 | 배송 사고 기록 — 둘 다 범인 아님 | 택배 대신 받아주며 웃는 장면 |
| neighbor-06 (수리비 분담) | 깨진 공용 시설 앞에 선 두 사람 | 견적서와 실제 금액 불일치 | 함께 청소하는 공용 공간 |
| neighbor-07 (이사 직후 첫 충돌) | 이삿짐 트럭과 복도의 긴장감 | 이전 거주자와의 과거가 소환 | 떡 돌리며 인사하는 새 이웃 |
| neighbor-08 (CCTV 갈등) | 현관에 설치된 CCTV 렌즈 클로즈업 | 편집된 영상의 프레임 분석 | CCTV 각도 조정하는 장면 |
| neighbor-09 (방음공사 합의) | 방음재가 쌓인 복도 | 서명한 합의서 vs 번복 문자 | 공사 완료된 벽에 손 대보는 장면 |
| neighbor-10 (아이들 갈등) | 놀이터에서 다투는 아이들 | 스티커가 앨범 안쪽에 있었다 | 아이들이 다시 뛰어노는 놀이터 |
| neighbor-11 (상가 공모) | 상가 건물 앞 보상 요구서 | 두 요구서가 같은 노트북에서 작성 | 건물주와 세 사람의 중재 테이블 |
| neighbor-12 (공동구매 분쟁) | 공동구매 단톡방 스크린샷 | 작년 미정산 62만원 — 진짜 문제 | 정산 완료 인증 메시지 |

### workplace 사건별 이벤트씬 제안

| 사건 | 인트로씬 | 반전씬 | 엔딩씬 |
|------|---------|--------|--------|
| workplace-01 | 인사평가 시즌, 늦은 사무실 불빛 | 문서 수정 로그 — 최종 수정자 위조 | 공정한 평가서를 제출하는 장면 |
| workplace-02 | 감사 직전 긴급 회의실 | 유출 경로 — 제3자가 진짜 유출자 | 진짜 유출자 징계 통보 |
| workplace-03 | 사내 메신저에 떠도는 소문 | 최초 발신자 추적 — 둘 다 아님 | 팀 회식에서 화해 |
| workplace-04 | 야근 거부 — 빈 사무실 | 초과근무 로그 vs 실제 업무량 | 근무 조건 재합의서 |
| workplace-05 | 발표장에서 떨리는 손 | 프레젠테이션 파일 수정 이력 | 공동 발표로 재조정 |
| workplace-06 | 접근 권한 경고 팝업 | 로그인 기록 — 권한 남용 경로 | IT 보안 교육 장면 |
| workplace-07 | 익명 게시판 캡처 | 게시글 편집 전후 비교 | 게시판 삭제 + 직접 대화 |
| workplace-08 | 법인카드 결제 알림 | 비공식 상계 합의서 존재 | 경비 규정 재정비 |
| workplace-09 | "공유본" 파일이 열린 화면 | 동명 문서 혼동 — 유출 아님 | 파일명 규칙 공지 |
| workplace-10 | 계약서 위의 정규직 전환 문구 | 이중 계약 — 숨겨진 조건 | 새 계약서 서명 |
| workplace-11 | 특허 출원서 앞의 두 이름 | 외부 브로커와의 메일 발견 | 공동 발명자 명기 |
| workplace-12 | 재택근무 화면 — 마우스 지글러 | 실제 업무 로그 vs 접속 로그 | 성과 기반 평가 전환 |

### 제작 방법 제안
- AI 이미지 생성 (Midjourney/DALL-E) → 톤 통일을 위해 일러스트 스타일 고정
- 실루엣/미니멀 스타일이면 제작 속도 빠르고 분위기 통일 가능
- 또는 텍스트+배경색 조합의 심플 카드 (이미지 없이)로 MVP
