# Thread S v3 Case Design: Spouse

### [spouse] 새벽 통화기록

**훅**:
남편 카드 명세서에 여성 생리대, 스타킹이 찍혀 있고, 퇴근 뒤엔 늘 같은 오피스텔에 두 시간씩 머문다.
게다가 새벽마다 같은 번호와 통화한다. 아내는 딴살림을 확신한다.

**A**: 박지연(36, 학원 데스크 직원) — 남편의 딴살림을 확신한 뒤, 이혼 대비로 집 돈 2,000만 원을 따로 빼두려다 투자 사기에 당했다 — `victim_cosplay`

**B**: 이준호(38, 가전매장 직원) — 딴살림이 아니라 친형과 중2 조카를 몰래 돌보고 있었고, 시댁과의 불화 때문에 입을 닫은 채 공동 적금 3,000만 원까지 깨서 형 빚을 갚아줬다 — `avoidant`

**callTerms**:
- A → B 기본: `여보`
- B → A 기본: `여보`
- A/B 격앙 시: `이준호!`, `박지연!`
- A → 재판관: `제 남편`
- B → 재판관: `제 아내`
- 규칙: 부부 사이 직접 호칭에 `~씨` 사용 금지

## 1. 쟁점 재배치

**초기 표시 쟁점**:
- d-1: 오피스텔 출입과 새벽 통화는 딴살림 흔적인가
  - 시작 complaint가 곧바로 꽂히는 표면 쟁점
  - 초반 Evidence Stub도 모두 이 의심을 강화하는 방향으로 보임
- d-2: 공동 적금 3,000만 원은 어디로 갔고, 배우자 동의 없이 움직인 것인가
  - 외도 의심이 단순 감정 문제가 아니라 가정 돈 문제로 번지는 쟁점

**숨은 쟁점**:
- h-d3: 아내의 2,000만 원 송금은 자기방어인가, 이혼 대비 은닉인가
  - 등장 조건:
  - `e-6`이 `Excerpt` 이상으로 상승하거나,
  - A에게 `동기탐색`이 2회 이상 누적되어 "내 돈이라도 지켜야 했다" 발언이 뜨면 탭 생성
- h-d4: 누가 먼저 신뢰를 깼는가
  - 등장 조건:
  - `h-d3`이 생성된 상태에서 `e-7`이 `Original` 이상이 되거나,
  - `e-5`와 `e-6`의 조합으로 `Reframe` Lead가 만들어지면 탭 생성

**쟁점별 앵커 진실**:
- d-1: 외도는 아니었지만, 숨길 만한 이유가 있는 가족 돌봄이었다
- d-2: 남편은 공동 적금 3,000만 원을 형 빚에 썼다
- h-d3: 아내도 딴살림 확신 뒤 2,000만 원을 따로 움직였다가 사기당했다
- h-d4: 숨김은 남편이 먼저 시작했지만, 실제 돈 이동은 아내가 먼저 실행했다

## 2. 증거 2축 설계

### e-1 [bank] 카드 명세서
- 역할: 외도처럼 보이는 가장 강한 표면 단서
- 깊이:
  - Stub: `카드 명세서 1건` 존재만 보임
  - Excerpt: 생리대, 스타킹, 컵죽, 즉석국 결제 항목만 보임
  - Original: 날짜, 시각, 금액, 편의점 지점이 모두 보이는 전체 카드 내역
  - Context: 4주 반복 패턴과 같은 날 `e-2`, `e-3`와 겹치는 점이 붙음
  - Established: `반복 구매 자체는 사실`로 공식기록 채택
- 신뢰 상태:
  - 제출됨: A가 캡처 제출
  - 검증 중: 재판관이 `원본 제출 명령`으로 카드앱 전체 내역 확인
  - 인증됨: 카드앱 원본과 은행 문자 시각이 일치
  - 이의 제기됨: B가 "품목만으로 외도 단정은 오독"이라고 주장
  - 조작/오독 판정: 데이터 자체는 인증되지만, `외도 증거`라는 해석은 이후 오독 판정 가능

### e-2 [log] 오피스텔 주차 출입 기록
- 역할: 반복 방문의 장소축 고정
- 깊이:
  - Stub: `오피스텔 출입기록 1건` 존재만 보임
  - Excerpt: 평일 저녁 같은 오피스텔에 반복 진입한 날짜 목록
  - Original: 차량 번호, 진입/출차 시각, 체류 시간까지 확보
  - Context: 같은 날 `e-1` 편의점 결제와 `e-3` 새벽 통화가 이어진 흐름이 붙음
  - Established: `B가 같은 오피스텔을 장기간 반복 방문한 사실`로 공식기록 채택
- 신뢰 상태:
  - 제출됨: A가 관리실 출력본 제출
  - 검증 중: 관리실 원본대장과 차량번호 대조
  - 인증됨: B 차량번호 일치
  - 이의 제기됨: B가 "외도 장소라고 단정할 수 없다"고 이의
  - 조작/오독 판정: 장소 데이터는 인증, 의미 해석은 열린 상태 유지

### e-3 [log] 새벽 통화기록
- 역할: 외도 서사를 완성시키는 시간축 단서
- 깊이:
  - Stub: `같은 번호와의 반복 통화` 존재 표시
  - Excerpt: 최근 3주 새벽 5시 전후 통화 횟수와 길이만 보임
  - Original: 전체 통화 일시와 번호, 통화 길이 확보
  - Context: 같은 날 저녁 `e-2` 방문이 이어지고, 이후 `e-1` 구매가 붙는 흐름이 보임
  - Established: `반복 통화와 반복 방문의 연동`이 공식기록에 채택
- 신뢰 상태:
  - 제출됨: A가 통화목록 캡처 제출
  - 검증 중: `원본 제출 명령`으로 통화 상세내역 대조
  - 인증됨: 통신사 상세내역과 일치
  - 이의 제기됨: B가 "통화 상대의 정체가 빠져 있다"고 맞섬
  - 조작/오독 판정: 통화 자체는 인증, 외도 해석은 `e-4` 전까지 잠정 가설

### e-4 [chat] 형 문자 + 조카 학교 알림 문자
- 역할: 1차 반전. 외도 오해를 돌봄 서사로 재맥락화
- 민감정보 봉인 대상:
  - 조카 이름
  - 학년/학교명
  - 생리 관련 직접 표현
- 깊이:
  - Stub: `같은 번호와의 문자 스레드` 존재만 보임
  - Excerpt: "오늘 혼자 있으니 밥 좀 부탁", "스타킹 하나만 사다 줘" 정도만 보임
  - Original: 연락처가 친형으로 확인되고, 학교 알림과 생필품 요청까지 연결됨
  - Context: "제수씨는 우리 집 얘기만 나오면 또 싸움 난다, 말하지 마" 문장과 형의 장시간 노동 맥락까지 복원
  - Established: `오피스텔 방문 대상은 형네였고, 숨김의 이유 중 하나는 시댁 불화`로 공식기록 채택 가능
- 신뢰 상태:
  - 제출됨: B가 발췌본으로 제출하거나, 재판관이 존재를 포착
  - 검증 중: `원본 제출 명령`으로 B 휴대폰 전체 스레드 확인
  - 인증됨: 연락처, 학교 알림, 날짜가 `e-2`, `e-3`와 부합
  - 이의 제기됨: A가 "형네 문자라고 해서 3,000만 원까지 정당화되진 않는다"고 이의
  - 조작/오독 판정: 조작은 없고, 오히려 `외도 서사`를 뒤집는 맥락 증거로 전환

### e-5 [bank] 공동 적금 해지 내역 + 형 계좌 이체
- 역할: 2차 반전의 핵심. 선의와 월권이 동시에 서는 금전 증거
- 깊이:
  - Stub: `공동 적금 해지 내역 1건` 존재만 보임
  - Excerpt: 해지일과 3,000만 원 금액만 보임
  - Original: 같은 날 형 계좌로 이체된 정확 내역까지 확인
  - Context: 형의 "내가 갚을게" 문자, 급한 빚 독촉 정황이 붙음
  - Established: `B가 배우자 동의 없이 공동 적금 3,000만 원을 형에게 보냈다`로 공식기록 채택
- 신뢰 상태:
  - 제출됨: A가 통장내역 일부 제출
  - 검증 중: 계좌 원본 조회 또는 은행 앱 원본 확인
  - 인증됨: 해지와 이체가 동일 날짜로 확인
  - 이의 제기됨: B가 "빌려준 돈"이라고 주장
  - 조작/오독 판정: 데이터는 인증, 사용 목적은 `Context`와 심문으로만 확정

### e-6 [chat/bank] A의 투자방 카톡 + 2,000만 원 송금 영수증
- 역할: 숨은 쟁점 생성. A도 돈을 움직였다는 사실 공개
- 깊이:
  - Stub: `A의 개인 송금 1건` 존재만 보임
  - Excerpt: 2,000만 원 송금 영수증과 "잠깐 넣어둘게" 문장만 보임
  - Original: "남편이 바람이면 내 돈부터 지켜야겠다" 대화, 투자방 링크, 송금 계좌 일치까지 보임
  - Context: 이후 답이 끊기고, A가 사기당한 뒤 숨긴 정황과 수치심이 복원됨
  - Established: `A가 이혼 대비 심리 속에서 2,000만 원을 따로 움직였다가 잃었다`로 공식기록 채택 가능
- 신뢰 상태:
  - 제출됨: B가 존재를 문제 삼거나, A가 일부만 제출
  - 검증 중: `분리심문(A)` + `원본 제출 명령`으로 전체 카톡과 송금내역 대조
  - 인증됨: 채팅 시각, 송금 시각, 계좌번호가 일치
  - 이의 제기됨: A가 "숨기려 한 건 맞지만 은닉은 아니었다"고 감정적 이의
  - 조작/오독 판정: 사기 피해 자체는 인증, `순수한 자기방어`인지 `이혼 대비 은닉`인지는 열린 해석 유지

### e-7 [timeline] 통화·출입·계좌 이동 대조표
- 역할: 상호 붕괴를 만드는 최종 시간축
- 깊이:
  - Stub: `비교 대조표 생성 가능` 상태만 표시
  - Excerpt: B의 비밀 방문 시작 시점과 A의 비밀 송금 시점의 거친 순서
  - Original: 통화, 방문, 적금 해지, 2,000만 원 송금의 정확한 날짜/시각이 한 장으로 정리됨
  - Context: 각 시점의 핵심 발언과 심리 상태가 덧붙어 "왜 그날 그 행동을 했는가"까지 읽힘
  - Established: `숨김은 B가 먼저 시작했고, 실제 비밀 송금은 A가 먼저 실행했다`는 시간축이 공식기록 채택 대상
- 신뢰 상태:
  - 제출됨: 없음. 조합과 검증을 통해 생성되는 종합 자료
  - 검증 중: `e-1`~`e-6` 중 핵심 축이 모두 인증될 때 자동 생성
  - 인증됨: underlying evidence 4개 이상이 인증되면 표도 인증
  - 이의 제기됨: A/B 모두 "순서가 의미 없다"며 이의 가능
  - 조작/오독 판정: 종합표 자체는 조작 불가, 다만 해석은 `Reframe` Lead 선택에 따라 달라짐

## 3. 조합 → Lead → Dossier 설계

### L-1 Timeline Lead: 같은 장소/반복 패턴
- 1차 조합 입력: `e-1 Excerpt` + `e-2 Excerpt`
- 생성 Lead: `Timeline`
- 플레이어 해석 선택지:
  - 해석 A: 외도 동선이다
  - 해석 B: 누군가를 돌보는 생활 동선이다
  - 해석 C: 단순 심부름일 수 있다
- 2차 조합: `L-1` + `e-4 Original`
- 생성 DossierCard: `dc-1 오피스텔의 사람들`
- 성공 효과:
  - d-1이 `외도 의심`에서 `형네 돌봄 여부` 쟁점으로 재정의됨
  - B의 회피 질감이 `회피`에서 `합리화` 단계로 진전
  - h-d4를 여는 첫 재맥락화 비트 발생

### L-2 Context Lead: 왜 아내에게 말하지 못했나
- 1차 조합 입력: `e-3 Context` + B의 발언 "가족 쪽 일입니다"
- 생성 Lead: `Context`
- 플레이어 해석 선택지:
  - 해석 A: 외도는 아니지만 창피해서 숨겼다
  - 해석 B: 시댁과의 불화를 피하려 숨겼다
- 2차 조합: `L-2` + `e-4 Context` 또는 `w-1 증언`
- 생성 DossierCard: `dc-2 시댁 얘기만 나오면 싸움`
- 성공 효과:
  - d-1에서 시댁 불화가 공식 서사로 떠오름
  - B의 `S3 전가`가 아니라 `S3 합리화` 성격이 선명해짐
  - 민감정보 봉인 해제를 할지 말지 플레이어가 고민하게 됨

### L-3 Beneficiary Lead: 3,000만 원은 누구를 살렸나
- 1차 조합 입력: `e-4 Established` + `e-5 Original`
- 생성 Lead: `Beneficiary`
- 플레이어 해석 선택지:
  - 해석 A: 형네를 실제로 살리려 한 돈이다
  - 해석 B: 형의 무책임을 대신 떠안은 월권이다
- 2차 조합: `L-3` + B의 발언 "형이 곧 갚는다고 했습니다" 또는 `w-1 증언`
- 생성 DossierCard: `dc-3 3,000만 원의 권한`
- 성공 효과:
  - d-2가 `합리화`에서 `자백` 직전까지 밀림
  - 평의에서 `금전 책임` 칸을 채울 수 있는 핵심 카드 생성
  - `선처 창구` Authority 행동의 효율이 크게 올라감

### L-4 Emotion Lead: 내 돈이라도 지켜야 했다
- 1차 조합 입력: `e-6 Excerpt` + A의 발언 "혹시 모르니까 제 몫을 지키려 했어요"
- 생성 Lead: `Emotion`
- 플레이어 해석 선택지:
  - 해석 A: 배신 공포 속 자기방어다
  - 해석 B: 이혼 대비 은닉이다
- 2차 조합: `L-4` + `분리심문(A)` 또는 `w-3 증언`
- 생성 DossierCard: `dc-4 2,000만 원의 수치`
- 성공 효과:
  - h-d3가 강제로 생성됨
  - A의 `victim_cosplay`가 약화되고 동기축 정보가 크게 열림
  - A가 투자 사기를 숨긴 이유까지 자발적으로 풀 수 있음

### L-5 Reframe Lead: 부부 돈 5,000만 원 증발
- 1차 조합 입력: `e-5 Established` + `e-6 Established`
- 생성 Lead: `Reframe`
- 플레이어 해석 선택지:
  - 해석 A: 남편이 더 큰돈을 먼저 무너뜨렸다
