# Thread W Korean Quality Checklist

## 목적
- GPT Pro 산출물을 **빠르게** 검수하기 위한 Thread W 전용 한국어 체크리스트다.
- 범위는 세 가지다.
  - 번역체 9패턴
  - 호칭 / 청자 규칙
  - Truth Throttle

## 1차 컷: 30초 금지어 스캔
- 아래 표현이 1개라도 보이면 일단 수정 대상으로 표시한다.
- `된 것으로 생각됩니다`
- `인 측면이`
- `부득이하게`
- `인지하고 있습니다`
- `에 기인`
- `해당 건에 대해서는`
- `하는 바입니다`
- `관련 사항을 간과`
- `여러 가지 상황이 얽혀`
- `사전 상의`
- `사전 협의`
- `미리 말씀드리지 못한`
- `특정 X`

## 번역체 9패턴 빠른 치환 기준

| 금지 표현 | 우선 치환 |
|---|---|
| `~된 것으로 생각됩니다` | `~인 것 같습니다` 또는 직접 단정 |
| `~인 측면이 있습니다` | `~한 점은 있습니다` |
| `부득이하게` | `어쩔 수 없이`, `급해서` |
| `인지하고 있습니다` | `알고 있습니다` |
| `~에 기인합니다` | `~ 때문입니다` |
| `해당 건에 대해서는` | `그 일은`, `거기에 대해서는` |
| `하는 바입니다` | 자연스러운 종결로 재작성 |
| `관련 사항을 간과` | `놓쳤습니다`, `빠뜨렸습니다` |
| `여러 가지 상황이 얽혀` | 구체적 상황 1개로 재작성 |

## 2차 컷: 호칭 / 청자 규칙

### 재판관 대상
- [ ] 재판관에게 말하는 문장은 모두 합니다체인가
- [ ] 재판관에게 상대를 말할 때 실명 대신 `callTerms.toJudge`를 썼는가
- [ ] 재판관 대상 문장 안에 `toPartner`, `angry` 호칭이 섞이지 않았는가

### 상대 직접 발화
- [ ] 상대에게 직접 말할 때만 `callTerms.toPartner` 또는 `callTerms.angry`를 썼는가
- [ ] direct clash 문장인데 재판관 존칭과 애칭이 한 문장에 같이 들어가지 않았는가
- [ ] 한 문장 안에 청자가 둘 이상 섞이지 않았는가

### 채널별 추가
- [ ] `witness`는 증인이 직접 말하는 문장인가
- [ ] `system_message`는 시스템 보이스인가
- [ ] `aftermath`가 판결 요약문이 아니라 후일담인가

## 호칭 빠른 판단표

| 상황 | 맞는 표현 | 틀린 표현 |
|---|---|---|
| 당사자 → 재판관에게 상대 언급 | `제 아내`, `제 남편`, 사건별 `toJudge` | 실명, `자기야` |
| 당사자 → 상대 직접 발화 | `자기`, 사건별 `toPartner` | `제 아내`, `제 남편` |
| 격앙된 직접 호명 | 사건별 `angry` | 평소 호칭과 섞은 혼합형 |

## 3차 컷: Truth Throttle

| 구간 | 허용 | 금지 |
|---|---|---|
| `S0-S1` | `해당 금액`, `그 사람`, `그곳`, 대략적 상황 묘사 | exact 금액, 실명, 기관 정식명칭, 정답 방향 직접 언급 |
| `S2` | `200만원대`, 성만 언급, 약칭 | full 실명, exact 금액, 기관 정식명칭, 구체 용처 |
| `S3` | 부분 인정, 일부 구체화 | 전체 진실 일괄 공개 |
| `S4-S5` | full 공개 가능 | 금전 사건 S5에서 숫자 없는 자백 |

## Truth Throttle 점검 질문
- [ ] S0에서 숨긴 사실을 S1에서 갑자기 뒤집지 않았는가
- [ ] 공개 범위가 단계적으로만 넓어지는가
- [ ] early state인데 숫자, 실명, 정식기관명이 튀어나오지 않는가
- [ ] 금전 사건 S5에 숫자가 1개 이상 들어가는가
- [ ] non-monetary 사건인데 돈/송금/계좌 같은 어휘가 튀어나오지 않는가

## 채널별 Truth Throttle 체크

### interrogation
- [ ] 첫 문장이 질문에 대한 직접 반응인가
- [ ] lieState가 올라갈수록 `defensive -> confident -> shaken -> angry -> resigned` 흐름이 보이는가

### evidence_present
- [ ] 증거에 반응하지 않고 정답을 먼저 풀어놓지 않는가
- [ ] 증거가 공개하지 않은 사실을 먼저 장황하게 설명하지 않는가

### dossier
- [ ] 질문이 결정적일수록 흔들리지만, 플레이어보다 먼저 정답을 선언하지 않는가

### witness
- [ ] `vague / partial / full` 구체성 차이가 분명한가

## 메타 / scaffold 냄새 체크
- [ ] `해당 쟁점`, `지금 질문하신`, `관련 사항`, `그 자료는` 같은 범용 scaffold가 과다하지 않은가
- [ ] 질문 원문, `truthDescription`, `surfaceName`, JSON 필드명이 대사에 그대로 붙지 않았는가
- [ ] 설명문, 브리핑문, 보고서문처럼 읽히지 않는가

## 최종 60초 판정
- [ ] 금지 표현 0건
- [ ] 재판관 대상 문장 합니다체 유지
- [ ] 청자 혼합 0건
- [ ] early-state 조기 누설 0건
- [ ] S5 금전 자백 숫자 포함
- [ ] 소리 내어 읽었을 때 실제 사람이 말하는 것처럼 들림
