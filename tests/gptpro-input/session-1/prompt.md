너는 솔로몬 법정 게임의 품질 분석 전문가다.

첨부된 파일은 spouse(부부) 카테고리 12건의 플레이스루 트랜스크립트야.
부부 사이의 비밀 송금, 외도 의심, 휴대폰 열람, 재산 은닉 등 부부 간 신뢰 파괴 사건들이야.

각 사건의 대화 로그를 분석해서 다음 5개 항목을 검증해줘.

---

## [검증 A: 화자 시점/호칭]

부부 사건에서 가장 흔한 오류 패턴:
- 재판관이 "제 아내에게", "제 남편이" 같은 NPC 1인칭 호칭을 사용 → **CRITICAL**
- 재판관은 반드시 "한지석 씨", "오세린 씨" 같은 3인칭 호칭을 써야 함
- 당사자가 재판관에게 배우자를 언급할 때: "제 아내가~", "제 남편이~" (toJudge 호칭) → 정상
- 당사자끼리 직접 말할 때: "자기야~", "오빠~" (toPartner 호칭) → 정상
- 조사 정합성: "제 아내**가**"(O), "제 아내**이**"(X) / "제 남편**이**"(O), "제 남편**가**"(X)
- 1인칭 혼동: 화자가 바뀌었는데 시점이 안 바뀌는 경우 → CRITICAL

**spouse 예시 (정상):**
- 재판관: "한지석 씨, 이 송금 건에 대해 설명해 주십시오." ✅
- 한지석→재판관: "제 아내가 걱정할까 봐 미리 말하지 못했습니다." ✅
- 오세린→재판관: "제 남편이 새벽에 보낸 메시지를 확인했습니다." ✅

**spouse 예시 (위반):**
- 재판관: "제 아내에게 송금하신 건에 대해..." ❌ CRITICAL (재판관이 NPC 시점 사용)
- 재판관: "남편분이 그러셨다는 건데..." ❌ (재판관은 이름+씨로만)

## [검증 B: 한국어 품질]

- 번역체 9패턴 검출:
  1. "~하는 것이 사실입니다" → "~한 건 맞습니다"
  2. "~에 대해 말씀드리겠습니다" → "~을 말씀드리겠습니다"
  3. "저는 ~라고 생각합니다" → "~라고 봅니다"
  4. "그것은 ~때문입니다" → "~때문입니다"
  5. "~하는 것은 아닙니다" → "~한 건 아닙니다"
  6. "~할 수 있는 상황이었습니다" → "~할 수 있었습니다"
  7. "~라는 점을 말씀드립니다" → "~입니다" (직접 진술)
  8. "상호 간의 합의" → "서로 합의" (한자어 과다)
  9. "해당 건에 대하여" → "그 건에 대해" (공문서체)
- "사전 상의/협의" 잔존 → FAIL (부부 사건에서 특히 주의. "미리 말하지 못한", "상의 없이" 등으로 대체되어야 함)
- 같은 표현 2턴 연속 반복 → WARN
- 어색한 조사/어미
- "~만을" 이중 조사

**spouse 자연스러운 표현 예시:**
- "적지 않은 돈을 미리 말하지 않고 보낸 건 제 잘못입니다." ✅
- "해당 금액에 대한 사전 협의가 부재했습니다." ❌ 번역체+공문서체

## [검증 C: Truth Throttle]

부부 사건의 진실 곡선: 비밀 송금 → 수취인 신원 → 실제 용도(간병/투자/채무)

- S0-S1: "적지 않은 돈", "그 사람" — 구체적 금액/인물명/기관명 절대 불가 → **CRITICAL**
- S2: "처가 쪽", "김 씨" 수준 — 성씨까지만, 대략적 금액만
- S3-S4: 점진적 구체화 ("장모님 쪽에 급한 일")
- S5: 전체 진실 공개 (금액+인물+기관+목적) — **반드시 구체적이어야 함**

**spouse 예시 (위반):**
- S1에서: "280만원을 최민정에게 보냈습니다" ❌ CRITICAL (S1에서 금액+인물)
- S5에서: "적지 않은 돈을 보냈습니다" ❌ FAIL (S5인데 여전히 모호)

---

## [검증 D: 캐릭터성/몰입감]

부부 사건 archetype 조합:
- avoidant(회피형): 우회적, "순서가 있어서..." "그 부분은..."
- confrontational(정면돌파형): 짧고 날카롭게, "오히려 ~가 먼저 아닙니까"
- cold_logic(냉정논리형): "사실관계를 정리하면..." 감정 배제
- victim_cosplay(피해자형): "저만 이런 취급을 받는 게..." 억울함

- 감정 변화 곡선: S0 방어/부정 → S2 불편/회피 → S3 blame/역공 → S4 감정적 무너짐 → S5 고백
- 이전 발언과 모순되는 내용 → FAIL
- tell 발현: 3~4턴에 1회 적절, 매턴 과다 → WARN

---

## [검증 E: Hallucination]

- 사건 데이터에 없는 인물/장소/날짜/금액 생성 → CRITICAL
- 부부 사건에서 특히: 실제 수취인이 아닌 다른 인물을 언급하는 경우
- 증거에 없는 내용을 "증거에서 봤다"고 말하는 경우

---

## [출력 형식]

사건별로:
```
{caseId}: PASS / FAIL
  CRITICAL: N건 (상세)
  FAIL: N건 (상세)
  WARN: N건 (상세)
```

마지막에 카테고리 종합:
```
총 12건 중 PASS: N건 / FAIL: N건
CRITICAL 총계: N건
가장 빈번한 문제 유형 TOP 3
```
